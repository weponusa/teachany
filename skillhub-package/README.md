# TeachAny（教我学）

> K12 各学科互动教学课件开发技能

## 简介

TeachAny 是一个面向 K-12 全学科的互动课件开发技能。它不是一个简单的 prompt 模板，而是一个**完整的教学设计系统**，将 6+ 套学习科学理论嵌入 AI 生成的课件中。

## 适用学科

数学、物理、化学、生物、地理、历史、语文、英语、信息技术（共 9 个学科）

## 核心能力

| 维度 | 说明 |
|:-----|:-----|
| **教学设计** | ABT 叙事结构、6 问预设计、课型分类（新授/复习/习题/专题/实验） |
| **学科适配** | 9 个学科各有专属讲解框架、互动组件和评估方式 |
| **互动课件** | 单文件 HTML，含 Canvas 仿真、拖拽排序、分步推导等 13 种互动组件 |
| **评估体系** | Bloom 六级认知分类、三级递进练习、错因诊断、量规评价 |
| **视觉设计** | 小学/初中/高中三套视觉模板，自动适配学段 |
| **视频增强** | 可选 Remotion 教学动画 + Edge TTS 语音讲解 + 双语字幕 |

## 理论基础

- ABT 叙事结构（Olson, 2015）
- 认知负荷理论（Sweller, 1988）
- Mayer 多媒体学习原则（Mayer, 2009）
- ConcepTest 同伴教学法（Mazur, 1997）
- Bloom 认知分类（Anderson & Krathwohl, 2001）
- 脚手架理论（Vygotsky, 1978; Wood et al., 1976）

## 原创方法论

- **五镜头法**（Five-Lens Method）：看见→拆开→解释→比较→迁移
- **学科适配矩阵**（Subject Adaptation Matrix）
- **6 问预设计框架**（6-Question Pre-Design）
- **课型分类体系** & **Phase 4 审查清单**
