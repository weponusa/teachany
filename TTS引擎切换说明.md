# TeachAny TTS 引擎切换说明

## 🎯 变更概要

**生效日期**：2026-04-14  
**版本**：v6.2.0

### 核心变更

**从**：Edge TTS（微软云端 TTS）  
**改为**：macOS 本地系统 TTS（`say` 命令）

---

## 📊 为什么要切换？

### Edge TTS 的问题

| 问题 | 影响 | 严重性 |
|:---|:---|:---:|
| **依赖网络** | 无网络环境无法生成语音 | ⚠️ 高 |
| **音质不稳定** | 24kHz 采样率，压缩损失 | ⚠️ 中 |
| **网络延迟** | 生成速度受网络影响 | ⚠️ 中 |
| **连接失败** | 网络波动导致生成中断 | ⚠️ 高 |

### macOS 本地 TTS 的优势

| 特性 | macOS say | Edge TTS |
|:---|:---:|:---:|
| **音质** | ⭐⭐⭐⭐⭐ 48kHz | ⭐⭐⭐ 24kHz |
| **速度** | ⭐⭐⭐⭐⭐ 即时 | ⭐⭐ 需网络 |
| **稳定性** | ⭐⭐⭐⭐⭐ 100% | ⭐⭐⭐ 70-90% |
| **离线** | ✅ 完全离线 | ❌ 必须联网 |
| **成本** | ✅ 免费 | ✅ 免费 |
| **API限制** | ✅ 无限制 | ✅ 无限制 |

## 🔧 技术实现

### 新增文件

```
scripts/
└── generate-tts-local.py   # 本地TTS生成脚本（新增）
```

### 脚本特性

**`generate-tts-local.py`**：
- ✅ 使用 macOS `say` 命令
- ✅ 自动检测系统语音
- ✅ 支持多种中英文语音
- ✅ 可选 ffmpeg 优化（转换为 MP3）
- ✅ 降级策略（无 ffmpeg 时输出 AIFF）

**使用示例**：
```bash
# 生成中文语音（默认 Tingting）
python3 scripts/generate-tts-local.py zh

# 生成英文语音（默认 Samantha）
python3 scripts/generate-tts-local.py en

# 列出可用语音
python3 scripts/generate-tts-local.py zh --list-voices

# 自定义语速
python3 scripts/generate-tts-local.py zh --rate 200
```

## 🚀 升级指南

### 用户端操作（一次性）

#### 1. 检查系统语音

```bash
# 列出已安装的语音
say -v ?

# 查找中文语音
say -v ? | grep zh

# 查找英文语音
say -v ? | grep en
```

#### 2. 下载高质量语音（推荐）

**macOS 设置路径**：
```
系统偏好设置
  → 辅助功能
    → 语音
      → 系统语音
        → 自定（点击"+"）
          → 选择"简体中文 - Tingting"下载
```

**预计大小**：约 300MB（高质量语音包）

#### 3. （可选）安装 ffmpeg 优化音质

```bash
# 使用 Homebrew 安装
brew install ffmpeg

# 验证安装
ffmpeg -version
```

**不安装 ffmpeg 的影响**：
- 输出为 AIFF 格式（文件较大，约 10MB/分钟）
- 功能完全正常，只是文件体积较大

## 📝 AI 工作流变更

### 旧流程（Edge TTS）

```
1. 检测 Python
2. 安装 edge-tts（pip install）
3. 编写旁白脚本
4. 执行 generate-tts.py
5. 生成 MP3 文件
```

**问题**：
- ❌ 步骤 2、4 需要网络
- ❌ 网络失败时课件不完整

### 新流程（本地 TTS）

```
1. 检测 macOS 系统
2. 检测 say 命令
3. 检查可用语音
4. 编写旁白脚本
5. 执行 generate-tts-local.py
6. 生成 MP3 文件（或 AIFF）
```

**优势**：
- ✅ 全程离线
- ✅ 100% 成功率（无网络依赖）
- ✅ 速度更快

## 🆘 常见问题

### Q1：我的 Mac 没有 Tingting 语音怎么办？

**A**：按以下步骤下载：
```
系统偏好设置 → 辅助功能 → 语音 → 系统语音 → 自定（+）
→ 选择"简体中文 - Tingting"→ 下载
```

### Q2：不安装 ffmpeg 可以吗？

**A**：可以！不影响功能，只是：
- 输出文件为 AIFF 格式
- 文件较大（约 10MB/分钟 vs 2MB/分钟）

### Q3：非 macOS 用户怎么办？

**A**：临时方案：
1. 继续使用 `generate-tts.py`（Edge TTS）
2. 或等待 Windows/Linux 支持（v6.3.0）

### Q4：Edge TTS 会被删除吗？

**A**：不会！`generate-tts.py` 保留，用户可手动使用。
