📚 串联并联特征对比

🔴 串联电路

R总 = R₁ + R₂ + R₃
总电阻等于各电阻之和
  • 电流：各处电流相等 I = I₁ = I₂
  • 电压：各部分电压之和等于总电压
  • 特点：一处断路，全路断电

🔵 并联电路

1/R总 = 1/R₁ + 1/R₂
总电阻小于最小分支电阻
  • 电压：各支路电压相等 U = U₁ = U₂
  • 电流：干路电流等于各支路之和
  • 特点：各支路互不影响
