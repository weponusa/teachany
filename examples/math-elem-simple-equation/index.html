📖 核心概念

📝 方程的概念

方程是含有未知数的等式。未知数通常用 x, y 表示。

x + 3 = 8
含未知数 x 的等式
💡 等式：左边 = 右边；方程：等式中含有未知数

🔑 解方程

解方程就是求使方程成立的未知数的值，也叫方程的解。

x + 3 = 8
x = 8 - 3 = 5

验证：5 + 3 = 8 ✓

📋 解方程的方法（等式性质）

➕ 减法
x + a = b
两边减a：x = b - a
➕ 加法
x - a = b
两边加a：x = b + a
✖️ 除法
ax = b
两边除a：x = b ÷ a
➗ 乘法
x ÷ a = b
两边乘a：x = b × a
💡 等式性质：等式两边同时加、减、乘、除同一个数（除数≠0），等式仍然成立。
