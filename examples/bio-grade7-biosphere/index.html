MODULE 01
生物的基本特征
6大特征 · 记忆锚点：营呼排应长遗
▼
0/3题
✓ 已掌握
And你知道身边的花草树木、猫狗鸟虫都是生物，石头、水、空气是非生物
But但你能说清楚"生物"和"非生物"到底有什么本质区别吗？机器人能动，岩石会生长，它们是生物吗？
Therefore所以我们需要掌握生物的6大基本特征，建立判断"是否是生物"的科学标准

🌱 营养

生物需要摄取营养物质来获取能量和构建自身。植物通过光合作用制造有机物，动物通过摄食获取。

摄取营养 光合作用

💨 呼吸

生物通过呼吸作用分解有机物，释放能量。即使植物在黑暗中也进行呼吸（不是光合作用）。

有氧呼吸 释放能量

🚽 排泄

生物将体内的代谢废物排出体外。动物通过汗液、尿液排泄；植物通过落叶排泄。

代谢废物 排出体外

⚡ 应激性

生物对外界刺激做出反应的能力。含羞草受触碰叶片合拢；人缩手躲避烫热物体。

外界刺激 做出反应

🌾 生长繁殖

生物能够生长发育并繁殖后代，使物种得以延续。

生长发育 繁殖后代

🧬 遗传变异

生物具有遗传性（亲代与子代相似）和变异性（后代不完全相同）。

遗传 变异
🔑

记忆锚点

营呼排应长遗

营（营养）呼（呼吸）排（排泄）应（应激性）长（生长繁殖）遗（遗传变异）

🤖 易错辨析：机器人能运动、能对外界做出反应，但它不能自主摄取营养、不能繁殖后代，因此机器人不是生物。钟乳石能"生长"，但它是无机物沉积，不是生物的生长，因此钟乳石不是生物。

MODULE 02
生态因素
非生物因素 vs 生物因素 · 可视化实验
▼
0/3题
✓ 已掌握
And你知道生物生活在一定的环境中，比如鱼离不开水，树需要阳光
But但影响生物生活的因素有哪些？同类生物之间为什么会竞争又会合作？
Therefore所以我们要系统学习非生物因素和生物因素，理解生物与环境的互动关系

☀️ 非生物因素

无生命的物理化学因素

光照 — 影响植物光合作用；深海无光植物无法存活
温度 — 影响动植物分布；鱼类是变温动物
水分 — 是生命活动的必要条件；荒漠生物稀少
土壤 — 影响植物的根系和生长
空气 — O₂供呼吸，CO₂供光合

🐾 生物因素

同种或不同种生物之间的关系

捕食 — 狼吃兔；蛙吃虫（不同种）
竞争 — 同区域的草木争夺阳光水分
合作 — 蚂蚁共同搬运食物（同种）
寄生 — 跳蚤寄生在狗身上；菟丝子寄生植物
共生 — 根瘤菌与豆科植物互利共存

记忆下列属于生物因素的是（　）
A阳光
B温度
C水分
D竞争
阳光、温度、水分都是非生物因素（无生命物质）；竞争是生物之间的关系，属于生物因素！
理解"大鱼吃小鱼，小鱼吃虾米"体现的生物因素关系是（　）
A捕食
B竞争
C合作
D寄生
捕食关系：不同种生物之间，一方以另一方为食。大鱼吃小鱼正是典型的捕食关系！
分析树林中的松鼠与橡树，松鼠以橡树果实为食。这对橡树来说是（　），对松鼠来说是（　）。
A有益；有益
B有害（被食）；有益（获食）
C有益；有害
D无影响；有益
捕食关系中，被捕食者（橡树）受到损害，捕食者（松鼠）获得能量，这是典型的"一方得益一方受损"的捕食关系！

MODULE 03
生物对环境的适应与影响
双向互动 · 经典案例分析
▼
0/3题
✓ 已掌握
And你知道沙漠里有骆驼，北极有北极熊，不同环境有不同生物
But但生物只是被动地"适应"环境，还是也会主动"改变"环境？
Therefore因此我们要认识生物与环境的双向关系：适应是被动应对，影响是主动改变

✅ 生物适应环境

生物的形态、结构、行为等特点与所在环境相适应，是长期自然选择的结果。

🐪 骆驼驼峰储存脂肪 → 适应干旱沙漠
🐋 鲸鱼用肺呼吸 → 水生哺乳动物的进化
🌵 仙人掌叶片变为刺 → 减少水分散失

↔️ 生物影响环境

生物的生命活动会改变周围环境的物理、化学和生物特性。

🌳 树木蒸腾 → 增加空气湿度
🐛 蚯蚓松土 → 改善土壤结构
🦫 河狸筑坝 → 改变水域环境

🎯 经典案例：点击展开详解

🐪

骆驼的适应

驼峰·眼睫毛·宽厚脚掌

🌵

仙人掌的适应

刺状叶·肉质茎储水

🐻‍❄️

北极熊的适应

白色皮毛·厚脂肪层

🌳

植树造林的影响

防风固沙·调节气候

骆驼适应沙漠环境：
① 驼峰储存脂肪，代谢产生水分和能量；② 长而密的眼睫毛防风沙；③ 宽厚脚掌适合在沙漠中行走；④ 体温变化幅度大，减少排汗。适应性越强，生存能力越强。
仙人掌适应干旱环境：
① 叶片退化为刺，减少蒸腾散水；② 肉质茎储存大量水分；③ 根系发达且浅表，快速吸收稀少降雨；④ 表皮厚且有蜡质层，防水分散失。
北极熊适应寒冷环境：
① 白色皮毛与冰雪环境融合（保护色）；② 厚实脂肪层隔热保温；③ 毛发中空，有隔热效果；④ 体型较大，减小相对表面积，减少散热。
植树造林对环境的影响：
① 树木蒸腾大量水分，增加大气湿度；② 防风固沙，保护土壤免受风蚀；③ 净化空气，吸收CO₂；④ 为其他生物提供栖息地，增加生物多样性。这体现了生物对环境的积极影响。

骆驼刺的形态特征是为了适应沙漠干旱缺水的环境，根系发达利于吸水，是生物适应环境的典型案例！

A环境影响生物
B生物适应环境
C生物影响环境
D生物之间的竞争
蚯蚓通过自身活动改变了土壤结构，这是生物对环境的主动改造，体现了生物影响环境！
评价以下哪个说法最准确地描述了生物与环境的关系？
A生物完全由环境决定，只能被动适应
B生物能完全改变环境，不受环境限制

生态系统的组成
生产者·消费者·分解者 · 食物链拖拽构建
▼
0/3题
✓ 已掌握
And你知道森林里有植物、动物、蘑菇和各种微生物
But但这些生物在生态系统中各自扮演什么角色？它们之间如何通过食物链传递能量？
Therefore所以我们要区分生产者、消费者和分解者，学会正确书写食物链
🌿

生产者

通过光合作用将无机物合成有机物，是生态系统的能量入口

草 树木 藻类 水草

主体：植物

🐾

消费者

直接或间接以生产者为食，不能自己制造有机物

兔（初级） 狐（次级） 鹰（顶级）

主体：动物

🍄

分解者

将有机物分解为无机物，归还到无机环境，促进物质循环

细菌 真菌 蚯蚓

主体：微生物

📌 食物链书写规则：①从生产者开始；②箭头表示物质和能量的流动方向（指向捕食者）；③只包括生产者和消费者（分解者不在食物链中！）。
正确示例：草 → 兔 → 狐狸 → 狼

MODULE 05
生物圈与生态系统
最大生态系统 · 知识图谱可视化
▼
0/3题
✓ 已掌握
And你已知道生态系统的组成：生产者、消费者、分解者和非生物环境
But但地球上有多少种不同的生态系统？它们之间有关联吗？有没有最大的生态系统？
Therefore因此我们要认识生物圈是地球上最大的生态系统，是所有生物共同的家园

🌍 生物圈的范围

地球表面上有生物生存的薄层，包括：
・大气圈下层（约15km高）
・岩石圈上层（约3km深）
・水圈全部

🌊 水生生态系统

包括海洋、湖泊、河流等。
特点：覆盖面积最大，种类多样

海洋生态系统 淡水生态系统

🏜️ 陆地生态系统

包括森林、草原、荒漠、农田、城市等。
森林生态系统——"地球之肺"

森林 草原 荒漠

🔬 生态系统的稳定性

生物种类越多，食物链越复杂，生态系统自我调节能力越强，稳定性越高。

种类多=稳定 自我调节

生物圈与各生态系统的包含关系

💡 重要结论：生物圈是地球上最大的生态系统，是所有生物共同的家园。它包含了地球上所有的生态系统，各生态系统之间通过物质循环和能量流动相互联系，共同维持着地球生命的存在。
