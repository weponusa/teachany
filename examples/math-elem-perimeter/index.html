📖 核心概念

📏 什么是周长？

周长是围成一个图形的所有边的总长度。

周长 = 各条边之和
💡 "周"表示一圈，"长"表示长度。周长就是绕图形走一圈的距离。

🔲 长方形周长

长方形有两条长和两条宽。

C = 2 × (长 + 宽)
C = 2长 + 2宽

例：长6cm，宽4cm
周长 = 2×(6+4) = 20cm

⬛ 正方形周长

正方形四边相等。

C = 4 × 边长

🔺 三角形周长

三角形三边之和。

C = a + b + c

⬠ 多边形周长

各边长度之和。

C = 各边之和
