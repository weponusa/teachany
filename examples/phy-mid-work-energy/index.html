📐 公式总结

• 功：W = Fs·cosθ（F：力，s：位移，θ：夹角）

• 功率：P = W/t = Fv·cosθ（v：速度 = s/t）

📖 核心概念

⚙️ 功（W）

力对物体做的功等于力与位移的乘积再乘以夹角的余弦：W = Fs·cosθ。单位：焦耳（J），1J = 1N·m。

核心概念

⚡ 功率（P）

单位时间内完成的功：P = W/t。描述做功的快慢。单位：瓦特（W），1W = 1J/s。

核心概念

✅ 做功条件

①有力作用在物体上；②物体在该力方向上发生位移。两者缺一不可，否则该力对物体做功为零。

基本条件

📊 正负功

力与位移同向（θ=0）：做正功；力与位移垂直（θ=90°）：做零功；力与位移反向（θ=180°）：做负功（阻力做负功）。

方向判断

🔋 额定功率

机械或电气设备正常工作时的功率。实际功率可以低于额定功率工作，但不应长时间超额运行。

工程概念

💡 功率与速度

恒力做功时：P = Fv（v为速度）。这说明发动机功率一定时，速度越大则牵引力越小，反之亦然。

综合应用

✅ 课堂练习

Q1：以下情形中，哪个力对物体做了功？

A. 举重运动员举着杠铃静止不动
B. 踢出去的足球在空中飞行（不计空气阻力）时，脚对球的力
C. 人推着小车在水平路面匀速前行
D. 圆形轨道上做匀速圆周运动时，向心力对物体
✅ C正确：人推小车有位移，推力方向与位移方向相同，做正功。A：有力无位移，不做功；B：球离手后脚已不接触球，脚的力不存在；D：向心力与运动方向垂直，不做功。

Q2：一台机器的功率是500W，工作了2分钟，做了多少功？

A. 1000 J
B. 250 J
C. 60000 J
D. 500 J
✅ W = P×t = 500W × 120s = 60000J。注意：时间单位要换算为秒（2分钟 = 120秒）！

Q3：用力F=100N沿水平方向推动重物，位移s=5m，力与位移成30°角，做了多少功？

A. 500 J
B. 250√3 ≈ 433 J
C. 250 J
D. 0 J
✅ W = Fs·cosθ = 100×5×cos30° = 500×(√3/2) ≈ 433J。cos30° = √3/2 ≈ 0.866。注意：不是所有力都全部用来做功，只有力在位移方向上的分量（F·cosθ）才做功。

Q4：甲、乙两台机器做相同的功，甲用时较短，则下列说法正确的是？

A. 甲做的功多
B. 甲的功率大
C. 乙的功率大
D. 两者功率相同
✅ P = W/t，W相同，甲用时短（t甲 < t乙），所以P甲 = W/t甲 > W/t乙 = P乙。甲的功率更大，说明甲做功更快、"本领"更强。功相同，效率（功率）不一定相同！
