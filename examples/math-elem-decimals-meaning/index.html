📖 模块一：小数是什么？

🌉 为什么需要小数？

我们已经知道……整数可以表示1米、2米、3米。

但有个难题……小明身高1米多一点，不到2米，怎么精确表达？

小数来帮忙！1.35米——这就是小数！

💡 小数的含义

小数是十进分数的另一种写法。

📐 把1米平均分成10份，1份是0.1米 = 1/10米（十分之一米）
📐 把1米平均分成100份，1份是0.01米 = 1/100米（百分之一米）
📐 把1米平均分成1000份，1份是0.001米 = 1/1000米

0.3 = 3/10    0.25 = 25/100    0.125 = 125/1000

🔢 小数各位的名称

以 3.456 为例：

• 3 在个位，表示3个1

• 4 在十分位，表示4个0.1（即4/10）

• 5 在百分位，表示5个0.01（即5/100）

• 6 在千分位，表示6个0.001（即6/1000）

🔍 生活中的小数

💰 价格：5.99元 = 5元9角9分
📊 气温：36.5°C（正常体温）
🏃 速度：100米用时9.83秒（博尔特世界纪录）

✏️ 马上试试

0.7 等于几分之几？

A. 7/100（百分之七）
B. 7/10（十分之七）✓
C. 7/1000（千分之七）

📖 模块二：小数的性质

💡 末尾添0去0，大小不变！

在小数的末尾添上0或去掉0，小数的大小不变。

0.5 = 0.50 = 0.500

⚠️ 注意：只有末尾可以添/去0，中间的0不能动！

0.50 → 去末尾0 → 0.5 ✓
0.50 → 去中间0 → 0.5（变成了5，不对！❌）

🔧 利用性质化简小数

0.7000 → 去末尾三个0 → 0.7

3.80 → 去末尾0 → 3.8

但 3.08 → 中间的0不能去！→ 还是 3.08

✏️ 化简小数

把 0.800 化简成最简形式：

A. 0.80
B. 0.8 ✓（末尾两个0都可以去掉）
C. 8（不对！把小数点也丢了）

📖 模块三：比较小数大小

💡 比较步骤

第一步：比较整数部分，大的那个小数更大

第二步：整数部分相同，比较十分位

第三步：十分位相同，比较百分位……依此类推

例：比较 3.14 和 3.09

整数部分都是3，相同 → 看十分位：1 > 0 → 所以 3.14 > 3.09

✏️ 比大小

比较 2.345 和 2.354：

A. 2.345 > 2.354
B. 2.345 < 2.354 ✓（百分位 4 < 5）
C. 2.345 = 2.354

⭐⭐⭐ 挑战题

0.1 和 0.10 是同一个数吗？

A. 是的！✓ 大小相等，只是写法不同
B. 不是，0.10更大因为位数多
C. 不是，0.1表示1位小数，0.10表示2位小数

📝 课程小结

✅ 小数性质：末尾添0或去0，大小不变
