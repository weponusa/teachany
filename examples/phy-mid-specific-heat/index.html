📖 模块一：为什么需要比热容？

🌉 沙滩和大海的温差

你知道… 夏天海边，沙滩晒得烫脚，但海水却相对凉爽。

但问题是… 太阳照射强度相同，为什么沙子比海水热得多？

所以要学… 不同物质吸收相同热量时，温度升高的幅度不同——这个"差异"由比热容决定。

💡 比热容的定义

比热容（c）是单位质量的某种物质，温度升高1°C（或1K）所吸收的热量。

单位：J/(kg·°C) 或 J/(kg·K)

比热容是物质的固有属性，与质量、温度变化量无关，只与物质种类和状态有关。

🔍 物理意义

水的比热容 c = 4.2×10³ J/(kg·°C)，意思是：每升高1°C，1千克的水需要吸收4200焦耳的热量。比热容越大，升温越难（储热能力越强）。

✏️ 即时练习

铁的比热容约为 0.46×10³ J/(kg·°C)，水的比热容为 4.2×10³ J/(kg·°C)。相同质量的铁和水吸收相同热量后：

A. 铁的温度升高更多，因为铁的比热容小
B. 水的温度升高更多，因为水的比热容大
C. 升高一样多，因为吸收热量相同

📖 模块二：热量公式 Q = cmΔt

💡 公式推导

实验发现，物体吸收（或放出）的热量与三个因素有关：

  • 质量 m 越大 → 吸收热量 Q 越多（正比）
  • 温度变化 Δt 越大 → 吸收热量 Q 越多（正比）
  • 比热容 c 越大 → 吸收相同温度变化需要更多 Q（正比）
Q = c · m · Δt
Q：热量（J）　c：比热容 J/(kg·°C)　m：质量（kg）　Δt：温度变化量（°C）
Δt = t末 - t初　　（吸热：Δt > 0；放热：Δt < 0）

📐 例题计算

将 2 kg 的水从 20°C 加热到 70°C，水吸收的热量为多少？

已知：c水 = 4.2×10³ J/(kg·°C)，m = 2 kg，Δt = 70 - 20 = 50°C

Q = c × m × Δt

Q = 4.2×10³ × 2 × 50

Q = 4.2×10⁵ J = 420000 J

🔬 深层理解

🌊 海陆风与比热容

海边白天：太阳加热陆地（沙石）比加热海水快，陆地温度高 → 陆地热空气上升 → 海风吹来

夜晚：陆地冷却快（比热容小），海水散热慢（比热容大）→ 海面温度高 → 陆风向海面吹

这就是海陆风的形成原因，也是为什么沿海地区昼夜温差比内陆小。

💡 迁移应用

暖气系统用热水而不是热空气：水的比热容是空气的约3500倍，相同质量的水携带的热量远多于空气，散热效果好得多。太阳能热水器也利用了水的大比热容储存能量。

📊 课堂总结

🗺️ 知识地图

  • 📏 比热容 c：物质固有属性，单位质量升温1°C所需热量，单位 J/(kg·°C)
  • 📐 公式：Q = c·m·Δt（Δt = 末温 - 初温）
  • 💧 水的 c 最大：4200 J/(kg·°C)，是优良冷却剂和储热介质
  • 🌊 海陆风：沙石比热容小（升温快），海水比热容大（升温慢）
  • 🚗 应用：汽车水冷、暖气热水、太阳能热水器
