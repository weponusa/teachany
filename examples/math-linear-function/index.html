📊 第一课：正比例函数 y = kx

📌 概念引入

弹簧挂重物实验：弹簧伸长量 y（cm）与所挂重物质量 x（kg）的关系：

x / kg01234
y / cm02468

观察发现 y = 2x，y 与 x 成正比例关系。

📝 定义

一般地，形如 y = kx（k 为常数，k ≠ 0） 的函数叫做正比例函数。

其中 k 叫做比例系数。

🔍 深层理解：为什么图像一定过原点？

因为 y = kx 中，当 x = 0 时 y = k·0 = 0，所以点 (0, 0) 永远在图像上。这正是"正比例"的本质——没有初始量，从零开始。弹簧不挂重物时伸长量为 0，行驶 0 分钟路程为 0。

🎬 正比例函数 y = kx 图像绘制

📐 图像特征

1
图像是一条直线，且一定经过原点 (0, 0)
2
k > 0 时，图像经过第一、三象限，y 随 x 增大而增大
3
k < 0 时，图像经过第二、四象限，y 随 x 增大而减小
k 的值经过象限增减性
k > 0一、三象限增函数 ↗
k < 0二、四象限减函数 ↘

📈 第二课：一次函数 y = kx + b

📌 从正比例函数到一次函数

某手机话费月租 10 元，每分钟通话 0.3 元。设通话 x 分钟，月费 y 元：

y = 0.3x + 10

这个函数比正比例函数多了一个常数项 +10，就是一次函数。

📝 定义

一般地，形如 y = kx + b（k, b 为常数，k ≠ 0） 的函数叫做一次函数。

当 b = 0 时，y = kx 即为正比例函数。所以正比例函数是一次函数的特殊情况。

🔍 深层理解：b 的本质是什么？

b 就是"初始值"或"起点"——手机月租、出租车起步价、水箱初始水量……现实中很多变化不是从零开始的，b 就代表这个"底"。从图像看，b 决定了直线与 y 轴的交点 (0, b)。改变 b 就是把整条直线上下平移。

🎬 一次函数 y = kx + b 概念引入

🖊️ 画图方法：两点法

一次函数图像是直线，两点确定一条直线，所以只需找两个特殊点：

1
令 x = 0，得 y = b → 点 (0, b)（与 y 轴的交点）
2
令 y = 0，得 x = -b/k → 点 (-b/k, 0)（与 x 轴的交点）
3
过这两个点画直线即可

🔍 第三课：斜率、截距与图像性质

📈 斜率 k 的几何意义
🔀 截距 b 的平移效果

📐 斜率 k 的意义

k = Δy / Δx = (y₂ - y₁) / (x₂ - x₁)
  • |k| 越大，直线越陡峭
  • k > 0：从左到右上升 ↗
  • k < 0：从左到右下降 ↘
  • k = 1 时与 x 轴成 45° 角

🔀 截距 b 的意义

b 是直线与 y 轴交点的纵坐标，即交点为 (0, b)

  • b > 0：直线向上平移
  • b = 0：经过原点（退化为正比例函数）
  • b < 0：直线向下平移

💡 改变 b 值相当于将 y = kx 上下平移 |b| 个单位

🗺️ k、b 的符号决定图像经过的象限

条件 经过象限 图像大致形状
k > 0, b > 0一、二、三↗ 上升，交 y 轴正半轴
k > 0, b < 0一、三、四↗ 上升，交 y 轴负半轴
k < 0, b > 0一、二、四↘ 下降，交 y 轴正半轴
k < 0, b < 0二、三、四↘ 下降，交 y 轴负半轴

💡 口诀：k 正左低右高，k 负左高右低；b 正上移，b 负下移

✍️ 典型例题精讲

📝 例题 1：判断正比例函数

下列哪些是正比例函数？哪些是一次函数？

① y = 3x    ② y = 2x + 1    ③ y = x²    ④ y = -5x    ⑤ y = 3/x    ⑥ y = -2x + 3

👁️ 查看解析
✅ 解析

正比例函数（形如 y = kx, k ≠ 0）：① y = 3x，④ y = -5x

一次函数（形如 y = kx + b, k ≠ 0）：① ② ④ ⑥

③ y = x² 是二次函数，不是一次函数
⑤ y = 3/x 是反比例函数，不是一次函数
正比例函数一定是一次函数，但一次函数不一定是正比例函数

📝 例题 2：求一次函数解析式（待定系数法）

已知一次函数 y = kx + b 的图像经过点 A(1, 5) 和点 B(-1, 1)，求该函数的解析式。

👁️ 查看解析
✅ 解析（待定系数法）
1
将 A(1, 5) 代入 y = kx + b：k + b = 5 … ①
2
将 B(-1, 1) 代入 y = kx + b：-k + b = 1 … ②
3
① + ②：2b = 6，b = 3
4
代入 ①：k = 5 - 3 = 2

∴ 一次函数解析式为 y = 2x + 3

📝 例题 3：图像与性质综合

一次函数 y = -2x + 4 的图像：

（1）与 x 轴、y 轴分别交于哪两个点？

（2）图像经过哪几个象限？

（3）当 x 取什么值时 y > 0？

👁️ 查看解析
✅ 解析

（1）

  令 x = 0：y = 4 → 与 y 轴交于 (0, 4)

  令 y = 0：-2x + 4 = 0 → x = 2 → 与 x 轴交于 (2, 0)

（2）

  k = -2 < 0，b = 4 > 0 → 经过一、二、四象限

（3）

  y > 0 即 -2x + 4 > 0 → x < 2

  ∴ 当 x < 2 时，y > 0

💡 这就是一次函数与一元一次不等式的联系！图像在 x 轴上方的部分对应 y > 0

📝 例题 4：实际问题建模

某蓄水池原有水 20 吨，打开进水管后每小时进水 5 吨。

（1）写出蓄水量 y（吨）与时间 x（小时）的关系式

（2）8 小时后蓄水池有多少水？

（3）蓄水池最多装 80 吨水，几小时后水满？

👁️ 查看解析
✅ 解析

（1） y = 5x + 20（x ≥ 0）

（2） x = 8 时：y = 5 × 8 + 20 = 60 吨

（3） y = 80 时：80 = 5x + 20 → x = 12 小时

🚀 提高题（中考真题改编）

🏆 提高题 1：两直线交点

已知一次函数 y₁ = 2x - 1 和 y₂ = -x + 5，求它们的交点坐标，并求当 x 取何值时 y₁ > y₂。

👁️ 查看解析
✅ 解析

求交点：联立 2x - 1 = -x + 5

  3x = 6 → x = 2 → y = 2(2) - 1 = 3

  ∴ 交点为 (2, 3)

求 y₁ > y₂：

  2x - 1 > -x + 5 → 3x > 6 → x > 2

💡 图像理解：x > 2 时，y₁ = 2x - 1 的图像在 y₂ = -x + 5 的上方

🏆 提高题 2：面积计算

一次函数 y = -¾x + 3 的图像与坐标轴围成的三角形面积是多少？

👁️ 查看解析
✅ 解析

令 x = 0：y = 3 → 交 y 轴于 A(0, 3)

令 y = 0：-¾x + 3 = 0 → x = 4 → 交 x 轴于 B(4, 0)

三角形的底 = OB = 4，高 = OA = 3

S = ½ × 4 × 3 = 6

🏆 提高题 3：含参数讨论

一次函数 y = (m-2)x + (m+1) 的图像不经过第二象限，求 m 的取值范围。

👁️ 查看解析
✅ 解析

首先，是一次函数需要 k ≠ 0，即 m - 2 ≠ 0 → m ≠ 2

不经过第二象限，分析：

  在第二象限中 x < 0, y > 0

  需要 k > 0 且 b ≤ 0（图像经过一、三、四象限或一、三象限）

即：

  m - 2 > 0 → m > 2

  m + 1 ≤ 0 → m ≤ -1

两个条件不能同时满足，所以考虑另一种情况——不过第二象限也可以是过原点且 k > 0：

  这需要 b = 0 且 k > 0 → m = -1 且 m > 2，矛盾。

重新分析：不经过第二象限的条件是 k > 0 且 b ≤ 0

由于 m > 2 和 m ≤ -1 不能同时成立，说明不存在满足条件的 m 值。

⚠️ 或者应是"不经过第三象限"才有解：k > 0 且 b > 0 → m > 2 且 m > -1 → m > 2

🏆 提高题 4：函数与不等式综合

如图，一次函数 y = kx + b 的图像经过 A(-3, 4) 和 B(2, -1)。

（1）求函数解析式；（2）求 y = kx + b 的图像与坐标轴围成的三角形面积；（3）根据图像直接写出 kx + b > 0 的解集。

👁️ 查看解析
✅ 解析

（1） 代入 A(-3, 4)：-3k + b = 4 … ①

  代入 B(2, -1)：2k + b = -1 … ②

  ② - ①：5k = -5 → k = -1 → b = 1

  ∴ y = -x + 1

（2）

  交 y 轴：(0, 1)，交 x 轴：(1, 0)

  S = ½ × 1 × 1 = ½

（3）

  kx + b > 0 即 -x + 1 > 0 → y > 0

  图像在 x 轴上方的部分对应 x < 1

  ∴ 解集为 x < 1
