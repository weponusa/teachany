🎯 引导 Engage

🤔 从生活问题出发

[And] 你已经学会了一次函数 y = kx + b，能画直线、求斜率、解决匀速运动问题。
[But] 篮球投篮的弧线不是直线，喷泉的水柱也不是直线——现实世界中很多轨迹是弯的，一次函数搞不定。
[Therefore] 所以我们要学一种新函数——二次函数，它的图象是一条优美的抛物线，能描述所有"先升后降"或"先降后升"的现象。

🏀 任务驱动：这个球最高能飞多高？

一个篮球从手中抛出，高度随时间变化的关系是：

h = -5t² + 20t

这里的 t² 让函数图象从直线变成了抛物线。

💡 学完今天的课，你就能用配方法一秒算出：最高点是 20 米，在 t = 2 秒时到达！

📋 生活中的二次函数

  • 正方形面积：S = x²（边长越大，面积增长越快）
  • 自由落体：h = ½gt²（物理课会用到！）
  • 利润最大化：定价太高没人买，太低没利润——存在最优价格
  • 桥梁拱形：抛物线是最优受力结构

→ 它们都有一个共同特征：含 x² 项！

📖 课时1：二次函数的概念

[And] 你知道 y = kx + b 叫"一次函数"，因为 x 的最高次数是 1。 [But] 如果出现了 x² 项，它还算一次函数吗？什么时候算？什么时候不算？ [Therefore] 这节课先搞清楚：什么才算二次函数——定义看起来简单，坑却不少。

定义

一般地，形如 y = ax² + bx + c（a ≠ 0） 的函数，叫做 x 的二次函数。

参数名称条件作用
a二次项系数a ≠ 0（关键！）决定开口方向和大小
b一次项系数可为 0影响对称轴位置
c常数项可为 0图象与 y 轴交点的纵坐标
⚠️ 易错点： 判断二次函数时，必须确保 a ≠ 0。例如 y = (m-1)x² + x 是二次函数的条件是 m ≠ 1。

📈 课时2：y = ax² 的图象

🎯 课时3：配方法与顶点式

❓ 为什么要配方？一般式有什么"缺陷"？

面对 y = 2x² - 8x + 5，你能直接回答以下问题吗？

❌ 顶点在哪？看不出来 ❌ 对称轴是？看不出来 ❌ 最值是多少？看不出来

但如果我们把它变成 y = 2(x - 2)² - 3（顶点式），一切都一目了然：

✅ 顶点 (2, -3) ✅ 对称轴 x = 2 ✅ 最小值 -3
💡 结论： 配方的目的 = 把一般式 y=ax²+bx+c 变成顶点式 y=a(x-h)²+k，让隐藏的信息变得一眼可见！

🤔 深层理解：为什么配方出来就是顶点式？

很多同学都有疑问：为什么 y = a(x-h)² + k 就叫顶点式？为什么配方配出来的就是顶点？下面从两个角度帮你彻底理解。

📐 角度一：几何直觉 —— 从"祖先"y = x² 出发

所有二次函数的"祖先"都是 y = x²，它的顶点在原点 (0, 0)。

↔ 水平平移
y = x² → 顶点 (0, 0)
y = (x-3)² → 右移3，顶点 (3, 0)
y = (x+2)² → 左移2，顶点 (-2, 0)
y = (x-h)² → 水平移 h，顶点 (h, 0)
↕ 垂直平移
y = (x-h)² → 顶点 (h, 0)
y = (x-h)² + k → 上下移 k
→ 顶点变为 (h, k)
 
最后乘 a 调开口：y = a(x-h)² + k

a 只改变开口大小和方向，不改变顶点位置，顶点始终是 (h, k)！

💡 一句话总结： 顶点式就是 "从原点出发，平移到哪里" —— h 是横坐标，k 是纵坐标。

🔢 角度二：代数理解 —— 为什么配方能"逼出"顶点？

以 y = x² + 4x + 1 为例，一步步看配方为什么恰好找到了顶点：

第1步：准备配方 → x² + 4x
第2步：加上 (4÷2)² = 4 → x² + 4x + 4 = (x+2)²
第3步：保持平衡，减去 4 → x² + 4x + 1 = (x+2)² - 4 + 1
第4步：整理 → y = (x+2)² - 3

关键洞察： (x+2)² ≥ 0 恒成立（平方数不可能为负）！
所以 y = (x+2)² - 3 ≥ -3。
当且仅当 x+2 = 0（即 x = -2）时，(x+2)² = 0，y 取到最小值 -3。
这个点 (-2, -3) 就是顶点！

📋 通用推导：从 y = ax² + bx + c 到顶点公式

y = a[x² + b/a·x] + c
= a[(x + b/2a)² - (b/2a)²] + c
= a(x + b/2a)² + c - b²/4a
顶点横坐标
h = -b/(2a)
注意负号！
顶点纵坐标
k = (4ac-b²)/(4a)
🎬 动画：为什么要配方 + 完整配方过程

📐 配方法四步详解（以 y = 2x² - 8x + 5 为例）

1 提系数：把二次项系数 a 提到括号外
y = 2x² - 8x + 5 → y = 2(x² - 4x) + 5

🔑 为什么：只有 x² 系数为 1，才能使用完全平方公式 (x+m)² = x² + 2mx + m²。所以先把 2 提出来，括号内 2x²÷2=x²，-8x÷2=-4x。

2 凑平方：一次项系数÷2再平方，加上它减去它
y = 2(x² - 4x + 4 - 4) + 5

🔑 关键推理：括号内一次项 -4x，取系数 -4，除以 2 得 -2，(-2)² = 4。加 4 减 4 = 加了个 0，值不变！但凑出了完全平方。

3 写平方：括号内用完全平方公式，展开括号外
y = 2(x - 2)² + 2 × (-4) + 5

🔑 两步操作：① x²-4x+4 恰好 = (x-2)²，写成平方形式；② 括号内剩下的 -4 要乘以外面的系数 2，得到 2×(-4) = -8，再加上原来的 +5。

4 算常数：化简得到顶点式 ✅
y = 2(x - 2)² - 3

🔑 化简：-8 + 5 = -3。大功告成！顶点 (2, -3)，对称轴 x=2，最小值 -3。

口诀：提 → 凑 → 写 → 算 ，每步都验算！

⚠️ 配方法三大易错点

错误类型❌ 常见错误✅ 正确做法
忘记提系数 y=2x²-8x+5 直接配方
→ 2(x-4)²+... 😱		 先提出2，括号内变成 x²-4x
→ 2(x²-4x)+5

（把整个系数平方了）		 -4÷2=-2，(-2)²=4

→ 2(x-2)²-4+5=3		 -4 要乘以外面的2
→ 2(x-2)²+2×(-4)+5=-3

🎯 顶点式的结构

y = a(x - h)² + k
一般式 → 顶点式（配方法）：
y = ax² + bx + c
= a(x² + b/ax) + c
= a(x + b/2a)² + c - b²/4a
= a(x + b/2a)² + 4ac - b²/4a

其中 h = -b/2a，k = (4ac-b²)/4a

📊 三大要素一览

要素公式含义
顶点(h, k) = (-b/2a, (4ac-b²)/4a)抛物线最高/最低点
对称轴x = h = -b/2a抛物线的对称线
最值ymin/max = ka>0 最小值，a<0 最大值
💡 快速记忆：
  • h 在括号里面，控制左右位置（注意取负号！）
  • k 在括号外面，控制上下位置
  • a 决定开口方向和宽窄

🔄 课时4：图象的平移变换

[And] 你知道 y = a(x-h)² + k 的顶点是 (h, k)，只要知道 h 和 k 就能定位顶点。 [But] 考试题经常问"把 y=x² 怎么移得到 y=(x-3)²+2？"，你需要把顶点变化翻译成平移语言。 [Therefore] 掌握平移口诀："左加右减、上加下减"——但要小心，这里的"加减"指的是顶点式里 h 和 k 的变化！

🔗 课时5：与一元二次方程的关系

[And] 你会解方程 ax²+bx+c=0，也会画函数 y=ax²+bx+c 的图象。 [But] 你有没有发现——"方程的根"和"图象与 x 轴的交点"好像是同一件事？ [Therefore] 本节揭示"三位一体"：方程的根 = 函数的零点 = 图象与 x 轴的交点，判别式 Δ 是连接它们的桥梁。

🎬 动画：二次函数与方程关系

核心联系：y = 0 时的 x 值 ↔ 方程的根 ↔ 图象与 x 轴的交点

判别式 Δ = b²-4ac方程 ax²+bx+c=0 的根图象与 x 轴
Δ > 0两个不相等实根 x₁, x₂两个交点 (x₁,0), (x₂,0)
Δ = 0两个相等实根（重根）一个交点（切点）
Δ < 0无实数根无交点

📝 典型例题

例1：判断二次函数

下列函数中，哪些是二次函数？
① y = 3x²   ② y = x² - 2x + 1   ③ y = 1/x²   ④ y = (x-1)² - x²   ⑤ y = ax² + bx + c

分析：判断是否为二次函数，关键看：① 是否为整式函数 ② x 的最高次数是否为 2 ③ 二次项系数是否不为 0

① ✓ 是，a=3, b=0, c=0

② ✓ 是，a=1, b=-2, c=1

③ ✗ 不是，1/x² = x⁻² 不是整式

④ ✗ 不是，展开 = x²-2x+1-x² = -2x+1，是一次函数

⑤ 不一定，需要 a ≠ 0 才是二次函数

例2：配方法求顶点

将 y = 2x² - 8x + 5 化为顶点式，并求出顶点坐标、对称轴和最值。

y = 2x² - 8x + 5

= 2(x² - 4x) + 5

= 2(x² - 4x + 4 - 4) + 5

= 2(x - 2)² - 8 + 5

= 2(x - 2)² - 3

顶点：(2, -3)

对称轴：x = 2

最值：a = 2 > 0，开口向上，当 x = 2 时 y 取最小值 -3

例3：图象平移

将抛物线 y = x² 经过怎样的平移，可以得到 y = (x-3)² + 2？

分析：比较 y = x²（顶点(0,0)）和 y = (x-3)²+2（顶点(3,2)）

h = 3 > 0 → 向右平移 3 个单位

k = 2 > 0 → 向上平移 2 个单位

答：先向右平移 3 个单位，再向上平移 2 个单位。

例4：利润最大化（中考真题改编）

某商品进价 30 元/件。市场调查：售价 x 元时，日销量为 (200-4x) 件 (30≤x≤50)。

① 写出日利润 y（元）与售价 x（元）的函数关系式；

② 当售价定为多少元时，日利润最大？最大日利润是多少？

① 建模：

每件利润 = x - 30

日销量 = 200 - 4x

y = (x - 30)(200 - 4x)

= -4x² + 320x - 6000

② 求最值：

y = -4(x² - 80x) - 6000

= -4(x - 40)² + 6400 - 6000

= -4(x - 40)² + 400

a = -4 < 0，开口向下，当 x = 40 在 [30, 50] 范围内

答：售价定为 40 元时，日利润最大为 400 元。

Level 2 能力应用（只给思路提示）

将 y = -3x² + 12x - 7 化为顶点式，求顶点和最值。

💡 提示：a = -3 ≠ 1，先提系数！括号内变成 x² - 4x。然后凑平方。注意 a < 0 说明开口向下，所以求的是最大值。

y = -3x² + 12x - 7

= -3(x² - 4x) - 7

= -3(x² - 4x + 4 - 4) - 7

= -3(x - 2)² + (-3)×(-4) - 7

= -3(x - 2)² + 5

顶点：(2, 5)　对称轴：x = 2　最大值：5（因为 a = -3 < 0）

Level 3 迁移挑战（独立设计）

设计题：请你写出一个二次函数，满足以下全部条件：

  • 开口向下
  • 顶点在第二象限
  • 经过原点

参考答案：y = -(x+2)² + 4

验证：① a = -1 < 0 → 开口向下 ✓

② 顶点 (-2, 4)，在第二象限（x<0, y>0）✓

③ 代入 x=0：y = -(0+2)²+4 = -4+4 = 0 ✓ 经过原点
