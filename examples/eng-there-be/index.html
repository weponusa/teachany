英语中，当我们想告诉别人"某处有某物"时，我们用：

There be + 某物 + 某处 "那里存在" → 某地有某物

🔑 最重要的一条规则

There be 后面跟的名词决定了用 is 还是 are：

后面的名词 be 动词 例子
单数（一个）
a book, a pen, one cat		 is There is a book.
不可数（数不清）
water, milk, bread		 is There is some water.
复数（多个）
two books, many cats		 are There are two books.
📝 口诀 单数、不可数 → 用 is
复数（两个及以上）→ 用 are

🔍 深层理解：为什么是 There be？

👁️ 看见它：中文说"有"，英文用 There be。这不是翻译，而是存在性表达——告诉别人某个地方"存在着"什么。
🔧 拆开它：There 是个引导词（没有实际意思），be 才是真正的谓语动词，随后面的名词变 is/are。就像一个"信号灯"，提醒后面要说"有什么"。
🌐 迁移它：以后学到 There was / There were（过去时）或 There will be（将来时），规则一样——只是 be 动词变了！

❌ 否定句：表示"没有"

在 be 后面加 not 就可以了！

肯定句 否定句 意思
There is a book. There is not a book.
(或 There isn't)		 没有一本书
There are two cats. There are not two cats.
(或 There aren't)		 没有两只猫
⚠️ 常见错误：不要说 "There no is..." 或 "There not is..."
正确写法是：There is not... 或 There isn't...

⁉️ 一般疑问句

把 be 动词提前到 there 前面，就变成了问句！

Be there ... ? → Yes, there be. / No, there be not.
陈述句 疑问句 回答
There is a book. Is there a book? ✓ Yes, there is.
✗ No, there isn't.
There are cats. Are there any cats? ✓ Yes, there are.
✗ No, there aren't.
