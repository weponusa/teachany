🌟 巧算妙法

买4件 25元 的玩具，总价 = 25 × 4 = 100元。但如果换成 (20+5)×4，可以拆开算：20×4+5×4=80+20=100。这就是分配律的魔法！掌握运算律，做题快如风！

📖 五大运算律
➕ 加法交换律
a + b = b + a
例：36 + 47 = 47 + 36 = 83
➕ 加法结合律
(a + b) + c = a + (b + c)
例：(28 + 72) + 35 = 100 + 35 = 135 ⚡更快！
✖️ 乘法交换律
a × b = b × a
例：25 × 4 = 4 × 25 = 100
✖️ 乘法结合律
(a × b) × c = a × (b × c)
例：(25 × 4) × 3 = 100 × 3 = 300 ⚡更快！
✖️ 乘法分配律
(a + b) × c = a×c + b×c
例：(20 + 3) × 5 = 20×5 + 3×5 = 100+15 = 115
