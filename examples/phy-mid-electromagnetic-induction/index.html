📖 模块一：电磁感应现象

🔮 法拉第电磁感应定律

电磁感应：导体（线圈）中的磁通量发生变化时，线圈中会产生感应电动势（感应电流）。

ε = -ΔΦ/Δt = -N·ΔΦ/Δt
ε：感应电动势(V) · N：匝数 · ΔΦ：磁通量变化量(Wb) · Δt：时间(s)
  • 产生感应电流的条件：磁通量变化（磁铁运动、线圈转动、电流变化等）
  • 磁铁静止在线圈内→磁通量不变→无感应电流
  • 磁通量变化越快（越快插拔），感应电动势越大
  • 匝数越多，感应电动势越大

📖 模块二：楞次定律

🔄 楞次定律

楞次定律：感应电流的磁场总是阻碍引起感应电流的磁通量的变化。

  • 磁通量增加 → 感应电流产生的磁场与原磁场方向相反（阻碍增加）
  • 磁通量减少 → 感应电流产生的磁场与原磁场方向相同（阻碍减少）
  • 右手定则：弯曲四指指向感应电流方向，大拇指指向该线圈内感应磁场方向

💡 楞次定律的本质

楞次定律是能量守恒定律的体现。若感应电流增强磁通量，则感应电流会"无中生有"地不断增大，违反能量守恒。所以自然界选择了"阻碍"的方向。

📖 模块三：发电机原理

⚡ 交流发电机

发电机利用电磁感应原理，将机械能转化为电能。

  • 线圈在磁场中旋转 → 磁通量不断变化 → 产生感应电动势
  • 线圈每转半圈，电流方向反转一次 → 产生交流电
  • 滑环取代换向器 → 保证产生交流电而非直流
  • 我国交流电：频率50Hz，每秒换向100次
