📖 模块一：分数 ↔ 小数

🌉 三兄弟，同一家！

我们已经知道……分数、小数、百分数都能表示一个量。

但有个难题……购物折扣写的是"打七五折"（75%），数学题里用分数，统计图用小数——同一件事要换来换去！

所以今天来学互化！让它们随意变身 🎭

💡 分数 → 小数

方法：分子 ÷ 分母

3/4 = 3÷4 = 0.75
1/8 = 1÷8 = 0.125
2/3 = 2÷3 = 0.666...= 0.6̄（循环小数）

💡 小数 → 分数

方法：按位值写分母，再化简

0.4 = 4/10 = 2/5（最简分数）
0.75 = 75/100 = 3/4
0.125 = 125/1000 = 1/8

✏️ 试一试

将 3/5 化为小数：

A. 0.6 ✓（3÷5=0.6）
B. 0.35（把分子分母拼起来了）
C. 1.67（算反了，5÷3）

📖 模块二：小数 ↔ 百分数

💡 小数 → 百分数

方法：小数点向右移两位，加上 % 号

0.75 → 75%
0.08 → 8%
1.2 → 120%（百分数可以超过100%！）

💡 百分数 → 小数

方法：去掉 % 号，小数点向左移两位

45% → 0.45
6% → 0.06
130% → 1.30 = 1.3

✏️ 小测验

0.038 化为百分数是：

A. 38%（移动了三位）
B. 3.8% ✓（移两位：0.038→3.8%）
C. 0.38%（只移了一位）

📖 模块三：分数 ↔ 百分数

💡 两步走

分数 → 百分数：先化成小数，再移小数点

3/4 → 0.75 → 75%

百分数 → 分数：百分数当分子，100当分母，再化简

60% = 60/100 = 3/5

🧠 深层理解

🌍 为什么要三种表达方式？

🛒 打折用百分数：打八折 = 80%，直观！

📐 精确计算用分数：1/3 比 0.333... 更准确！

📊 数据统计用小数：平均分3.75，小数最方便！

💡 三种形式各有所长，要根据场景选择！

🔍 思维拓展

超市大减价：原价100元，现价75元。
折扣 = 75/100 = 3/4 = 0.75 = 75%
→ 打七五折 = 七折半 = 75%优惠

📝 课程小结

🌟 互化公式总结

✅ 分数→小数：分子 ÷ 分母

✅ 小数→分数：按位值写，再化简

✅ 小数→百分数：小数点右移2位 + %

✅ 百分数→小数：去%，小数点左移2位

✅ 分数↔百分数：借助小数作跳板
