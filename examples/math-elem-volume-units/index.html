📖 核心概念
🔬

mm³（立方毫米）

边长1mm正方体的体积

沙粒、药丸级别
🎲

cm³（立方厘米）

边长1cm正方体的体积
= 1mL

糖块、药片级别
📦

dm³（立方分米）

边长1dm正方体的体积
= 1L = 1000cm³

牛奶盒、鱼缸级别
🏠

m³（立方米）

边长1m正方体的体积 = 1000dm³

房间、游泳池级别

📊 体积单位换算

1m³ = 1000dm³
1dm³ = 1000cm³ = 1L
1cm³ = 1000mm³ = 1mL
从大到小乘以从小到大除以
m³ → dm³×1000dm³ → m³÷1000
dm³ → cm³×1000cm³ → dm³÷1000
cm³ → mm³×1000mm³ → cm³÷1000
💡 体积单位每相邻两级之间的进率是1000（长度是10，面积是100，体积是1000）
