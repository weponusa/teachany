🫀 消化道——食物的旅行通道

消化道是一条长约9米的管道，食物从口腔进入，沿途被消化，最终残渣从肛门排出。

① 口腔
牙齿切割研磨食物；舌搅拌；唾液腺分泌唾液（含唾液淀粉酶），开始消化淀粉。
② 咽
食物通道，同时也是呼吸道的一部分；吞咽时会厌软骨盖住气管入口。
③ 食道
长约25cm；靠蠕动将食物推向胃；本身不分泌消化液，无消化功能。
④ 胃
容积约1.5L；胃腺分泌胃液（含盐酸+胃蛋白酶），开始消化蛋白质；胃壁蠕动将食物变成食糜。
⑤ 小肠
长约6m，是消化吸收的主要场所；接受胆汁、胰液、肠液，完成三大营养物质的消化；内壁有大量皱襞和绒毛增大吸收面积。
⑥ 大肠
长约1.5m；吸收水分和无机盐；将食物残渣形成粪便，推向肛门。
⑦ 肛门
消化道末端，粪便排出体外的出口。

🧪 消化腺与消化液

消化腺分泌消化液，其中含有消化酶，负责将大分子营养物质分解为小分子。

消化腺 分泌的消化液 主要成分/作用 作用部位
唾液腺 唾液 唾液淀粉酶 → 分解淀粉 口腔
胃腺 胃液 盐酸 + 胃蛋白酶 → 分解蛋白质 胃
肝脏 胆汁 不含消化酶，乳化脂肪（物理消化） 小肠
胰腺 胰液 胰蛋白酶 + 胰脂肪酶 + 胰淀粉酶 小肠
肠腺 肠液 多种消化酶，完成三大营养物质消化 小肠

📊 三大营养物质消化过程

营养物质 开始消化部位 参与消化的酶 最终产物 消化完成部位
淀粉 口腔 唾液淀粉酶 → 胰淀粉酶 → 肠液 葡萄糖 小肠
蛋白质 胃 胃蛋白酶 → 胰蛋白酶 → 肠液 氨基酸 小肠
脂肪 小肠 胆汁乳化 → 胰脂肪酶 → 肠液 甘油+脂肪酸 小肠

📌 本节核心知识小结

🚇
消化道
口腔→咽→食道→胃→小肠→大肠→肛门，全长约9米
🧪
消化腺
唾液腺/胃腺/肝脏/胰腺/肠腺，分泌消化液分解营养
🌟
小肠地位
消化吸收的主要场所，接受胆汁+胰液+肠液三大消化液
🔬
三大营养
淀粉→葡萄糖；蛋白质→氨基酸；脂肪→甘油+脂肪酸
🌿
胆汁特点
不含消化酶，由肝脏产生，乳化脂肪（物理消化）
💡
蠕动原理
消化道肌肉收缩舒张产生蠕动波，推送食物前进
