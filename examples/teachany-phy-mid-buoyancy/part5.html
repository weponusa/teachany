📖 模块三：物体的浮沉条件

🤔 什么决定浮还是沉？

[And] 我们知道 F浮 = ρ液gV排，G物 = ρ物gV物。
[But] 木块浮在水面、铁块沉入水底、鸡蛋在盐水中悬浮——同样是在水里，为什么结果截然不同？
[Therefore] 关键是比较 F浮 和 G物 的大小。对于实心均匀物体，本质就是比较 ρ物 和 ρ液！

💡 核心知识：三种浮沉状态

力的关系密度关系运动状态V排 关系生活实例
F浮 < G物 ρ物 > ρ液 下沉 V排 = V物 铁块在水中、石头沉底
F浮 = G物 ρ物 = ρ液 悬浮 V排 = V物 鸡蛋在适当盐水中
F浮 > G物 ρ物 < ρ液 上浮 → 漂浮 漂浮时 V排 < V物 木块浮在水面、冰山

⚠️ "悬浮"与"漂浮"的区别

悬浮：物体完全浸没在液体中且静止，ρ物 = ρ液，V排 = V物。可以在液体内部任意深度停留。

漂浮：物体浮在液面上且静止，ρ物 < ρ液，V排 < V物。一部分露出液面。

两者都满足 F浮 = G物，但 V排 不同！

🚢 漂浮问题的特殊性

漂浮时 F浮 = G物，这是解漂浮类题目的核心等式。

推导：漂浮时 ρ液gV排 = ρ物gV物
→ V排/V物 = ρ物/ρ液
→ 浸入体积占总体积的比例 = 密度之比

例：冰的密度 ≈ 0.9×10³ kg/m³，水密度 = 1.0×10³ kg/m³
→ V排/V物 = 0.9/1.0 = 90%
→ 冰山有 90% 体积在水下，只露出 10%——"冰山一角"由此而来！

🔧 改变浮沉的方法

让物体上浮

• 减小 ρ物：空心化（船舶设计）

• 增大 ρ液：加盐（死海浮人）

• 增大 V排：展平形状（铝箔实验）

让物体下沉

• 增大 ρ物：注水压舱（潜水艇）

• 减小 ρ液：从盐水换成淡水

• 减小 V排：排气缩小体积

🌍 生活中的浮沉应用

🚢 轮船：钢铁空心化 → 排开大量水 → F浮 > G → 漂浮。载货越多，吃水线越深。

🐟 潜水艇：压载水舱注水/排水 → 改变自身重力 → 下潜/上浮/悬浮。

🎈 热气球：加热空气 → 气球内空气密度 < 外部 → 浮力 > 重力 → 升空。

🏊 游泳：吸气时肺部膨胀 → V排增大 → 浮力增大 → 更容易浮。

🧊 冰山：ρ冰 < ρ水 → 90% 在水下 → 航海大忌。

🥚 盐水浮蛋：加盐增大 ρ液 → 鸡蛋从沉底到悬浮到漂浮。
