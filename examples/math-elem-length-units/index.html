📐 五大长度单位
💉

毫米 mm

1mm ≈ 一根头发丝的直径
药片厚度约 2mm
✏️

厘米 cm

1cm = 10mm
大拇指宽约 1cm
📏

分米 dm

1dm = 10cm = 100mm
手机长约 1.5dm
🚪

米 m

1m = 10dm = 100cm
教室门高约 2m
🛣️

千米 km

1km = 1000m
跑道约 0.4km一圈

🔄 单位换算口诀

  • 📌 1千米 = 1000米，记住"千"就是×1000
  • 📌 1米 = 10分米，手指张开约1dm
  • 📌 1分米 = 10厘米，尺子上10格
  • 📌 1厘米 = 10毫米，尺子最小刻度
