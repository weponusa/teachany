🧭 本课程的教学主张

古典诗词不是"阅读理解"——它有自己的语言系统：平仄、押韵、对仗、起承转合。教这些的最好方式，不是把它们系统地灌下去，而是把每个知识点分散在不同的诗里，一首诗承载一两个知识点，在反复诵读中自然习得。

本课程把系统知识拆成五章，每章锚定一个主线：诵读 / 平仄 / 押韵对仗 / 意象 / 诗体。学完之后，学生不是"背下了一堆术语"，而是真的能读懂一首陌生的古诗。

PRE-TEST

先导测试（Pre-test）

3 道题 · 定位起点
Pre-1

下列哪首作品是"律诗"？

《静夜思》——床前明月光……
《春望》——国破山河在，城春草木深……
《渔歌子》——西塞山前白鹭飞……
《将进酒》——君不见黄河之水天上来……
解析：律诗为八句四联，中间两联（颔联、颈联）必须对仗。《静夜思》四句是绝句；《春望》八句且"感时花溅泪，恨别鸟惊心"为对仗联——这是五言律诗。《渔歌子》是词，《将进酒》是古体诗。
Pre-2

朗诵"白日依山尽"时，哪种节奏更符合古诗诵读传统？

白日/依山尽（2-3，按意义停顿）
白日/依山/尽（2-2-1，两字一顿）
白/日/依/山/尽（逐字停顿）
白日依山尽（一气呵成不停顿）
解析：古诗的声音节奏遵循"两字一顿"：五言 2-2-1，七言 2-2-2-1。这和按语义分的意义节奏不同步——这正是诵读的核心技法。
Pre-3

"大漠孤烟直，长河落日圆"中的"孤烟"主要营造了什么意境？

凄凉哀伤——因为"孤"代表孤独
辽阔壮美——烟衬托出广袤沙漠与开阔视野
思乡怀人——孤独的烟柱让人想起故乡
无法判断
解析：意象不是固定标签。"孤"字在这里不是"孤独哀伤"，而是与"大漠""长河"配合营造辽阔壮美的边塞意境——这正是第四章的核心观念。
CHAPTER 1

诵读入门 · 声音节奏与意义节奏

约 20 分钟
And你读过很多古诗，大多按"意思"来停顿断句。
But这样读会把五言、七言都读成散文，丢掉诗的音律节奏感。
Therefore要先区分"声音节奏"和"意义节奏"——古诗的声音节奏有自己的规则：两字一顿。

🎵 两字一顿：古诗的声音骨架

古诗的诵读停顿不按意思分，按声音分：

  • • 五言诗：2-2-1 节奏（白日/依山/尽）
  • • 七言诗：2-2-2-1 节奏（两个/黄鹂/鸣翠/柳）

加上"平长仄短韵脚延"——平声字拖长、仄声字短促、韵脚字延长——你就还原了古诗本来的声音样貌。

登鹳雀楼
唐 · 王之涣
白日 · 依山 · 尽
黄河 · 入海 · 流
■ 平声拖长 · ■ 仄声短促 · ■ 韵脚延长

🔍 深度理解（Five-Lens）

👁 看一看 See

把五言切成 2/2/1 三块："白日 / 依山 / 尽"。你读出的不是意思，而是节奏。

🧩 拆一拆 Break

"白日依山尽"的意义是"太阳落在山后面"——按意思该 2-3 切。但声音节奏要求 2-2-1，两者不一致，这正是古诗的特色。

⚖ 对一对 Compare

现代朗诵腔（Rap、播音腔、篱笆墙式"43 分段"）读不出古诗的味道。古诗的节奏是独立于现代汉语朗诵的另一套系统。

❌ 常见错误读法

白日依山尽 / 黄河入海流（一口气读完，无停顿无拖腔）

或："白日 / 依山尽 / 黄河 / 入海流"（按意义节奏）

✅ 正确诵读示范

白日— / 依山— / 尽↗
黄河↘ / 入海— / 流↗↗

2-2-1 停顿 + 平声拖长 + 韵脚延长

📣 诵读方式速查

方式适用场景注意事项
齐读课堂开头的基本训练、气氛缓解、关键处高潮营造不宜贯穿全课
示范读核心环节，标注平仄与拖腔教师范读或 TTS 领读
比赛读激发诵读热情"敢不敢和老师比？"
古版读竖排无标点呈现，增趣又助记有难度的诗适用，太简单的诗不宜用
吟唱高级形式，诗歌情感的深度表达⚠ 未入门不宜上台，"抹黑"胜于增辉
Level 1Bloom 2 · 理解

七言诗"两个黄鹂鸣翠柳"的声音节奏应该是？

两个黄鹂 / 鸣翠柳（4-3 按意义）
两个 / 黄鹂 / 鸣翠 / 柳（2-2-2-1）
两个 / 黄鹂鸣 / 翠柳（2-3-2）
两 / 个 / 黄 / 鹂 / 鸣翠柳
解析：七言统一 2-2-2-1。意义上"两个黄鹂"是一个主语，但诵读时要在"两个"和"黄鹂"之间停顿——这是声音节奏的要求。
Level 2Bloom 3 · 应用

下列诵读方式，哪一个不适合在课堂使用？

齐读——全班一起读，用于课堂伊始或气氛缓解
示范读——教师范读或 TTS 音频领读
古版读——竖排无标点反复诵读
刚入门就上台吟唱——"吟不出味道就搬上讲台一定是抹黑"
解析：吟唱是诵读的高级形式，必须先吟出味道再上台；未入门就表演，效果适得其反。齐读/示范读/古版读都是经过验证的有效方式。

📌 本章要点

  • 古诗有两种节奏：声音节奏（两字一顿） vs 意义节奏（按意思分），两者不同步。
  • 平长仄短韵脚延——古诗诵读的三大声音特征。
  • 诵读方式按场景分：齐读（开头/气氛）、示范读（核心）、比赛读（激发）、古版读（增趣）。
  • 吟唱是高级形式，功底到位再上台。
CHAPTER 2

平仄与格律 · 声调的规则

约 25 分钟
And你能按 2-2-1 节奏诵读了。
But为什么"平声拖长、仄声短促"？什么是平仄？
Therefore本章认识四种标准格律句式，学会给一首诗标注平仄。

📐 平仄是什么

汉语有四声。古汉语：平、上、去、入。其中平声为"平"，上、去、入为"仄"。

古汉语声调现代普通话对应平仄归属
平声阴平（1 声）、阳平（2 声）平
上声上声（3 声）仄
去声去声（4 声）仄
入声（普通话已消失，分化入各声）仄

⚠ 入声字是难点——"绝、灭、雪、月"古为入声（仄），普通话却读成了平声。字典会标明入声，遇到时需特别记忆。

🔖 四种标准格律句式

格律诗每句的平仄有严格规律，归纳为四种基本句式：

类型五言七言
仄起仄收仄仄平平仄平平仄仄平平仄
平起平收平平仄仄平仄仄平平仄仄平
平起仄收平平平仄仄仄仄平平平仄仄
仄起平收仄仄仄平平平平仄仄仄平平

"一三五不论，二四六分明"——第 2、4、6 位字平仄严格，第 1、3、5 位相对灵活。

🔍 深度理解

👁 看一看 See

"白日依山尽"——仄仄平平仄（白=仄入声，日=仄入声，依=平，山=平，尽=仄）。这是"仄起仄收"。

⚖ 对一对 Compare

绝句 4 句、律诗 8 句、古体诗不限且不讲平仄。《静夜思》《登鹳雀楼》是格律绝句，《将进酒》是古体诗。

🚀 迁一迁 Transfer

拿一首你熟的绝句，试试给每个字标平仄。你会发现大多符合四种句式之一。

Level 2Bloom 4 · 分析

《静夜思》"床前明月光"的平仄模式是？

平平平平平（全平）
平平平仄平（床平、前平、明平、月仄、光平）
仄仄平平仄
平仄平仄平
解析：床（阳平）、前（阳平）、明（阳平）、月（古入声·仄）、光（阴平）。"月"是关键：普通话 4 声，但古为入声，归仄。《静夜思》其实并不严格守格律，属古体诗风格。

📌 本章要点

  • 平仄由声调决定：阴平/阳平为平；上声/去声/入声为仄。
  • 入声字是古汉语特有的，普通话中已消失。
  • 格律诗有四种基本句式，律诗八句由它们组合而成。
  • "一三五不论，二四六分明"——关键位置严格。
CHAPTER 03

押韵与对仗

约 22 分钟

🧭 本章脉络（ABT）

AND古人写诗讲究押韵，让诗读起来琅琅上口；律诗还要求颔联、颈联对仗。
BUT学生常弄不清"押什么韵、押在哪、用哪个韵书"，更搞不清对仗为什么要"词类相同、平仄相反"。
THEREFORE本章用四条押韵规则 + 两条对仗要求，建立可操作的判断清单。

🎵 押韵四规则

规则说明举例
① 位置押在偶数句末字二、四、六、八句尾
② 首句可押可不押五言多不入韵，七言多入韵《春望》首句押，《登高》首句押
③ 一韵到底全篇同一个韵部，不可换韵杜甫《登高》全押"灰"韵
④ 押平声韵律诗、绝句原则上押平声押仄声多见于古体诗

💡 判断口诀：偶句必押、首句看体、韵脚平声、一韵到底。

⚖️ 对仗两要求

① 词类相同

名词对名词，动词对动词，数词对数词……
例："两个黄鹂鸣翠柳，一行白鹭上青天"
两（数）对一（数）、黄鹂（名）对白鹭（名）、翠柳（名）对青天（名）。

② 平仄相反

上下联对应位置的字，平仄相反。
例："无边落木萧萧下，不尽长江滚滚来"
无（平）—不（仄）、落（仄）—长（平）、萧萧（平平）—滚滚（仄仄）。

📘 律诗规定：颔联（3、4 句）与颈联（5、6 句）必须对仗；绝句不强制。

🔍 五镜头·对仗赏析

赏析对象：王维《山居秋暝》颔联"明月松间照，清泉石上流"

  1. 看：月亮照在松林间，清泉流过石头上——两幅静/动画面并置。
  2. 拆：明月（名）/ 清泉（名）、松间（名）/ 石上（名）、照（动）/ 流（动）——词类一一对应。
  3. 解：上联写"视觉之静"，下联写"听觉之动"，动静结合营造幽远意境。
  4. 比：对比杜甫"两个黄鹂鸣翠柳"——同为对仗，但王维重"意境营造"，杜甫重"色彩鲜活"。
  5. 迁：写景时尝试"上联静态、下联动态"的对仗结构，如"夕阳山外落，归鸟林中栖"。
Level 2 · Bloom 应用

下列哪一组是合格的对仗？

A. "红花开满地，春风吹又生"
B. "春蚕到死丝方尽，蜡炬成灰泪始干"
C. "我爱祖国山河美，他们也爱这片土地"
D. "月亮圆了，星星亮了"
B ✓ 李商隐《无题》颔联。春蚕对蜡炬（名/名）、到死对成灰（动补/动补）、丝方尽对泪始干（词类与平仄双对应），是对仗典范。

📌 本章小结

  • 押韵四规则：偶句押、首句看体、一韵到底、平声为主。
  • 对仗两要求：词类相同 + 平仄相反。
  • 律诗颔联、颈联必须对仗。
CHAPTER 04

意象与意境

约 25 分钟

🧭 本章脉络（ABT）

AND诗歌最动人之处，是用有限的"象"（具体事物）唤起无限的"意"（情感与哲思）。
BUT学生读诗常止步于"翻译字面意思"，忽略意象在中国文化中的多义性与象征传统。
THEREFORE本章建立"意象辞典 + 起承转合"两套工具，从字面走向意境。

🏮 常见意象辞典

意象典型含义出处示例
月思乡、团圆、孤独、时间流逝"举头望明月，低头思故乡"
柳离别、挽留（"柳"谐"留"）"客舍青青柳色新"
杜鹃/子规哀思、亡国、思归"杨花落尽子规啼"
梅傲骨、高洁、不屈"俏也不争春，只把春来报"
孤舟 / 扁舟漂泊、孤独、隐逸"孤舟蓑笠翁"
夕阳 / 落日迟暮、乡愁、壮美"夕阳无限好，只是近黄昏"
大雁信使、思乡、秋意"雁字回时，月满西楼"

⚠️ 意象具有多义性——同一个"月"，可以是思乡、可以是孤独、可以是永恒。需结合上下文判断。

🪜 起承转合·四步读诗法

不是"总—分—总"，而是古人写诗的标准思路。

起｜引入

交代场景、人物、时间。
例：春眠不觉晓

承｜展开

顺着起句铺陈描写。
例：处处闻啼鸟

转｜转折

视角、时空或情感的转变。
例：夜来风雨声

合｜收束

升华、点题或留白。
例：花落知多少

🎯 孟浩然《春晓》——起承转合的教科书。"夜来风雨声"是关键"转"，把春日闲适忽然拉到惜春伤春。

🔍 五镜头·意境体悟

赏析对象：马致远《天净沙·秋思》

枯藤老树昏鸦，
小桥流水人家，
古道西风瘦马。
夕阳西下，
断肠人在天涯。
  1. 看：28 个字勾勒 11 种意象——枯藤、老树、昏鸦、小桥、流水、人家、古道、西风、瘦马、夕阳、断肠人。
  2. 拆：前三句全是名词短语，无动词、无连词；第四句 4 字过渡；末句点题。
  3. 解：冷色（枯、老、昏、瘦）包裹温色（小桥流水人家），反衬游子思乡之切。"断肠"二字把前面所有意象的情感能量聚焦释放。
  4. 比：对比王维"大漠孤烟直"——同写荒凉，王维走壮美路线，马致远走凄清路线。意象选择决定意境走向。
  5. 迁：尝试用 9 个名词意象写一句"春日思念"：嫩柳、新燕、细雨、湿阶……
Level 3 · Bloom 分析

"月落乌啼霜满天，江枫渔火对愁眠"——"月"这一意象在此最贴近哪种含义？

A. 团圆的喜悦
B. 永恒的时间
C. 孤寂与羁旅之愁
D. 爱情的思念
C ✓ 张继《枫桥夜泊》。"月落"是时间（已近拂晓仍未眠）+"乌啼、霜满、愁眠"构成孤寂羁旅氛围。这体现意象的多义性——必须结合上下文判断。

📌 本章小结

  • 意象是"具体事物+文化约定含义"的组合，具多义性。
  • 起承转合是古人写诗的标准结构，关键在"转"。
  • 冷暖意象并置、反衬、聚焦，是营造意境的常用手法。
CHAPTER 05

诗体辨析

约 18 分钟

🧭 本章脉络（ABT）

AND中学课本里的古诗，分属"古体诗、绝句、律诗、词"等不同诗体。
BUT学生常把《静夜思》当律诗、把《春江花月夜》当绝句，诗体概念模糊。
THEREFORE本章用"三看"辨体法——看句数、看字数、看格律，一分钟判定诗体。

📚 主要诗体对比表

诗体句数每句字数格律要求经典例
古体诗不限四/五/七言混合不严格，可换韵《春江花月夜》《茅屋为秋风所破歌》
五言绝句4 句5 字讲究平仄、押韵王之涣《登鹳雀楼》
七言绝句4 句7 字讲究平仄、押韵杜牧《山行》
五言律诗8 句5 字平仄+押韵+颔颈对仗王维《山居秋暝》
七言律诗8 句7 字平仄+押韵+颔颈对仗杜甫《登高》
词依词牌定长短句依词牌"填"苏轼《水调歌头·明月几时有》

⚡ 三看辨体法

  1. 一看句数：4 句 → 绝句系；8 句 → 律诗系；其他 → 古体或词。
  2. 二看字数：每句整齐 5 字或 7 字 → 近体诗（绝句/律诗）；长短不一 → 词或古体。
  3. 三看格律：8 句且颔颈对仗 → 律诗；4 句近体 → 绝句；句数不限、可换韵 → 古体。

💡 词还要多看一步："词牌名"决定了字数、句数、押韵。《如梦令》≠《念奴娇》。

Level 2 · Bloom 应用

李白《静夜思》"床前明月光，疑是地上霜。举头望明月，低头思故乡。" 属于哪种诗体？

A. 五言律诗
B. 五言绝句（古绝）
C. 七言绝句
D. 词
B ✓ 4 句、每句 5 字，是五言绝句。但此诗未严格遵守近体平仄（"明月光—地上霜—望明月—思故乡"押平声但平仄不严），属于"古绝"一类。

📌 本章小结

  • 三看辨体：句数 → 字数 → 格律。
  • 近体诗（绝句/律诗）格律严格；古体诗自由；词依词牌。
  • 律诗颔颈必须对仗，是辨识律诗的最快特征。

POSTTEST

课程后测

5 题 · 检验学习成效
Q1 · 诵读

五言诗"国破山河在，城春草木深"的标准节奏应是？

A. 国破山／河在，城春草／木深
B. 国破／山河／在，城春／草木／深
C. 国／破山河／在，城／春草木／深
B ✓ 五言诗标准 2-2-1 节奏。
Q2 · 平仄

普通话中，下列哪一组字全部是"仄声"？

A. 天、山、人、花
B. 月、雪、客、笑
C. 春、风、明、来
B ✓ 月（入→仄）、雪（上→仄）、客（入→仄）、笑（去→仄），均为仄声。
Q3 · 对仗

下列哪项不是严格意义上的对仗？

A. "海内存知己，天涯若比邻"
B. "两个黄鹂鸣翠柳，一行白鹭上青天"
C. "劝君更尽一杯酒，西出阳关无故人"
D. "无边落木萧萧下，不尽长江滚滚来"
C ✓ 此句出自王维《送元二使安西》，是"流水对"或干脆不对仗——上下句在语意上连贯推进，不符合"词类相同、平仄相反"的严格对仗。
Q4 · 意象

柳永"杨柳岸晓风残月"中的"柳"主要寄托哪种情感？

A. 春日生机
B. 离别不舍（柳—留）
C. 隐逸之志
D. 家国忧思
B ✓ "柳"在古典诗词中的典型含义即"挽留、离别"，取"柳"与"留"的谐音。
Q5 · 诗体

杜甫《登高》"风急天高猿啸哀……百年多病独登台。艰难苦恨繁霜鬓，潦倒新停浊酒杯。"共八句七言，且颔颈对仗。它是——

A. 七言绝句
B. 七言律诗
C. 七言古体诗
D. 词
B ✓ 八句 + 每句七字 + 颔颈对仗 = 七言律诗。被誉为"古今七律第一"。

🎊 结课寄语

古典诗词的学习不止于"考点"。当你能用节奏读出声韵之美、用格律看清文字秩序、用意象走进诗人内心、用诗体认清时代脉络——诗就不再是负担，而是一扇随身携带的美学之门。
