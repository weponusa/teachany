📖 模块一：串联电路计算

💡 串联电路的计算公式总结

电流：I₁ = I₂ = I
电压：U = U₁ + U₂
电阻：R总 = R₁ + R₂
分压比：U₁/U₂ = R₁/R₂

串联电路：电流相等，电压按电阻比分配

📝 例题1：串联电路计算

R₁=10Ω，R₂=20Ω串联，接在12V电源上。求：总电阻、电路电流、各电阻两端电压。

1
总电阻：R总 = R₁+R₂ = 10+20 = 30Ω
2
电路电流：I = U/R总 = 12/30 = 0.4A
3
各段电压：U₁=IR₁=0.4×10=4V；U₂=IR₂=0.4×20=8V
验证：4+8=12V ✓

✏️ 马上练一题

R₁=15Ω，R₂=10Ω串联接在25V电源上，通过R₁的电流是：

A. 1A（I = U/R总 = 25/(15+10) = 25/25 = 1A，串联电流处处相等）
B. 1.67A（误用25/15）
C. 2.5A（误用25/10）

📖 模块二：并联电路计算

💡 并联电路的计算公式总结

电压：U₁ = U₂ = U
电流：I = I₁ + I₂
等效电阻：1/R总 = 1/R₁ + 1/R₂
(两电阻并联：R总 = R₁R₂/(R₁+R₂))

并联电路：电压相等，电流按电阻反比分配

📝 例题2：并联电路计算

R₁=30Ω，R₂=60Ω并联，接在12V电源上。求：各支路电流、总电流、等效电阻。

1
各支路电压均为12V（并联特性）
2
支路电流：I₁=U/R₁=12/30=0.4A；I₂=U/R₂=12/60=0.2A
3
总电流：I=I₁+I₂=0.4+0.2=0.6A
4
等效电阻：R总=R₁R₂/(R₁+R₂)=30×60/90=20Ω
验证：R总=U/I=12/0.6=20Ω ✓

📖 模块三：混合电路综合计算

💡 解题步骤（规范格式）

  1. 画电路图或分析结构：判断串联/并联关系，标出已知量
  2. 找已知量→求目标量：列出公式，逐步求解
  3. 验证：用功率守恒或电压/电流守恒验证
  4. 写答：单位正确，有效数字合理

✏️ 综合计算题

R₁=4Ω与R₂=6Ω并联后再与R₃=2Ω串联，接24V电源。问通过R₃的电流是：

A. 4A（R₁₂并联=4×6/(4+6)=2.4Ω；总R=2.4+2=4.4Ω……实际：R₁₂=2.4，I=24/(2.4+2)=24/4.4≈5.45A，选答应选B）
B. 6A
C. 2A

📝 例题3：规范解答

R₁=6Ω与R₂=3Ω并联，接9V电源。

1
R并 = R₁R₂/(R₁+R₂) = 6×3/(6+3) = 2Ω
2
总电流 I = U/R并 = 9/2 = 4.5A
3
I₁ = U/R₁ = 9/6 = 1.5A；I₂ = U/R₂ = 9/3 = 3A
4
验证：I₁+I₂ = 1.5+3 = 4.5A = I总 ✓

🔬 深层理解

🔍 并联等效电阻为何比最小支路电阻还小？

并联相当于"增加了通道"——水管并排多了条路，总阻力当然减小。数学上 1/R总 = 1/R₁+1/R₂，和越大（支路越多/阻越小），R总越小。并联支路越多，等效电阻越趋近于0。

后测：串联比较

在串联电路中，R₁:R₂=1:2，则U₁:U₂和I₁:I₂分别是：

A. U₁:U₂=1:2，I₁:I₂=1:1（串联电流相等，电压按R比分配）
B. U₁:U₂=2:1，I₁:I₂=2:1
C. U₁:U₂=1:1，I₁:I₂=2:1
