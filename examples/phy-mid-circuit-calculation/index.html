📖 模块一：串联电路规律

🔗 串联电路三大规律

电流：I = I₁ = I₂ = I₃
串联电路中各处电流相等（只有一条通路）
电压：U = U₁ + U₂ + U₃
总电压等于各电阻两端电压之和（分压）
电阻：R = R₁ + R₂ + R₃
总电阻等于各电阻之和（等效电阻增大）

✏️ 串联计算

串联电路中，R₁=3Ω，R₂=7Ω，电源电压U=10V。求电路中的电流I和R₂两端电压U₂。

A. I=1A，U₂=7V
B. I=2A，U₂=14V
C. I=1A，U₂=3V

📖 模块二：并联电路规律

🔀 并联电路三大规律

电流：I = I₁ + I₂ + I₃
干路电流等于各支路电流之和（分流）
电压：U = U₁ = U₂ = U₃
各支路两端电压相等（等于电源电压）
1/R = 1/R₁ + 1/R₂ + 1/R₃
等效电阻的倒数等于各支路电阻倒数之和（等效电阻减小）

✏️ 并联计算

并联电路，R₁=6Ω，R₂=3Ω，电源电压U=6V。求总电阻R和总电流I。

A. R=9Ω，I=0.67A
B. R=2Ω，I=3A
C. R=2Ω，I=1A

📖 模块三：混联电路计算方法

📐 解题步骤

  1. 画出电路图，标出已知量
  2. 判断连接方式（串联/并联）
  3. 计算等效电阻（并联用倒数公式，串联直接相加）
  4. 用欧姆定律 I = U/R 求总电流
  5. 利用各部分关系求分支电流/电压

🔑 关键口诀

串联分压不分流，并联分流不分压 — 串联中阻值大的分得电压多；并联中阻值小的分得电流多！

✏️ 综合计算题

两电阻R₁=4Ω与R₂=8Ω并联后，再与R₃=2Ω串联，总电源U=12V。求总电流I₃和通过R₁的电流I₁。

A. I₃=4A，I₁=2A
B. I₃=2A，I₁=4/3 A ≈ 1.33A
C. I₃=3A，I₁=1A
