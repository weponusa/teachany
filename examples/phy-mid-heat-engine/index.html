📖 模块一：热机的工作原理

🌉 从蒸汽机到内燃机

你知道… 汽车发动机能驱动车轮转动，它需要汽油——燃料的化学能最终转化为车轮的机械能。

但问题是… 化学能→内能→机械能，这个过程是怎么实现的？为什么汽车还会发热？

所以要学… 热机——将内能转化为机械能的装置，以四冲程内燃机为典型。

💡 四冲程内燃机工作过程

口诀：吸 → 压 → 做 → 排（每个冲程活塞往返一次）

1
吸气冲程：活塞下移，进气门打开，吸入混合气（汽油蒸气+空气） 无能量转化，准备工作
2
压缩冲程：活塞上移，两气门关闭，压缩混合气 机械能 → 内能
3
做功冲程：火花塞点火，混合气爆炸，推动活塞下移 内能 → 机械能 ✨唯一做功冲程
4
排气冲程：活塞上移，排气门打开，排出废气 排出废热

四个冲程完成一个循环，曲轴转两圈，只有做功冲程是内能→机械能。

📖 模块二：热机效率与能流图

💡 热机效率的定义

η = W有用 / Q总
η：热机效率　W有用：有用功（J）　Q总：燃料燃烧放出的总热量（J）
各类热机效率：蒸汽机6~15%，汽油机25~30%，柴油机30~45%

🌊 热机能流图

输入总热量 = 有用功 + 废热（热力学第一定律）

🔍 为什么效率不能100%？

热力学第二定律：热量自发地从高温传向低温，不可能将热量全部转化为功而不产生其他影响。热机必然向外散失废热，效率永远小于100%。这是物理规律，无法用技术手段克服，只能尽量减少损耗。

📖 模块三：能量守恒定律

💡 能量守恒——自然界的铁律

能量守恒定律：能量不会凭空产生，也不会凭空消失；它只能从一种形式转化为另一种形式，或从一个物体转移到另一个物体，总量不变。

  • 热机：化学能 → 内能 → 机械能（+ 废热）
  • 发电机：机械能 → 电能
  • 电灯：电能 → 光能 + 热能
  • 总原则：所有形式的能量之和不变

💡 永动机（不消耗能量持续做功的机器）违反能量守恒定律，不可能存在。

🔬 深层理解

🚗 提高热机效率的方法

🔧 技术手段

  • • 提高燃烧温度和压强
  • • 减少摩擦（润滑、材料改进）
  • • 回收废热（废气余热发电）
  • • 混合动力（制动能量回收）

🌱 环保意义

  • • 效率每提升1%，燃料消耗显著下降
  • • 减少CO₂排放，缓解温室效应
  • • 新能源汽车（电动车）理论效率更高

💡 热力学第二定律

热量不能自发地从低温流向高温（需外界做功）。这是方向性规律，说明自然过程有方向性——已经混合均匀的气体不会自发分开，摔碎的杯子不会自动复原。热机效率永远小于100%正是这条定律的体现。

📊 课堂总结

🗺️ 知识地图

  • ⚙️ 热机：内能→机械能，以四冲程内燃机为典型
  • 🔄 四冲程：吸气→压缩（机→内）→做功（内→机）→排气
  • 📊 热机效率：η = W有用/Q总，各类热机效率不同，均小于100%
  • 🌊 能流图：Q总 = W有用 + Q废热
  • ⚡ 能量守恒：能量不灭，只转化；永动机不可能
  • ❄️ 热力学第二定律：热量自发从高温→低温，过程有方向性
