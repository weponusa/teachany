📖 模块一：小数点从哪里来？

🌉 整数和分数之间的桥梁

我们有整数（1, 2, 3...）和分数（1/2, 1/4...）……（And）

但是超市里的价格写的是 3.5 元，温度计显示 36.5——这是什么写法？（But）

这就是"小数"！是分数的另一种更简便的写法！（Therefore）

💡 小数点是"分界线"

3.5

← 整数部分   小数点   小数部分 →

小数点（那个小圆点）是"分界线"：
左边是整数部分（3 = 3个一）
右边是小数部分（5 = 5个十分之一 = 0.5）
合起来：3 + 0.5 = 3.5

🔢 用格子理解 0.1

把 1 平均分成 10 份，每份就是 0.1（零点一）：

0.1 = 1/10（十分之一）；0.5 = 5/10 = 1/2（二分之一）

📖 模块二：小数的读写方法

💡 小数怎么读？

读小数分两步：

第一步：整数部分按整数读
第二步：小数点读"点"，小数部分每位数字逐一读出

例：3.65
→ 整数部分 "三"
→ 小数点 "点"
→ 小数部分 "六五"
→ 读作：三点六五

🔍 小数的数位

小数点右边的数位从左到右依次是：
第一位（十分位）：0.1 = 1/10
第二位（百分位）：0.01 = 1/100
第三位（千分位）：0.001 = 1/1000
小数点左边是整数家族，右边是小数家族！

📖 模块三：比较小数大小

💡 小数比大小的步骤

比较两个小数的大小，按顺序：

第一步：先比整数部分
第二步：整数相同，比十分位
第三步：十分位相同，比百分位……以此类推

例：比较 3.7 和 4.2
整数：3 < 4 → 所以 3.7 < 4.2

例：比较 2.8 和 2.3
整数相同（都是2）→ 十分位：8 > 3 → 所以 2.8 > 2.3

⚠️ 易错点：小数位数不同时

误区：小数位数越多越大 ❌
正确：要对位比较，0.3 > 0.25（3 > 2，哪怕0.25有两位小数）

🔍 小数末尾的0

小数末尾加0或去0，大小不变：
0.5 = 0.50 = 0.500（末尾加0不改变大小）
这是小数的性质！但读数时有区别（0.50读"零点五零"）

🎯 比大小

比较 1.6 和 1.49：

A. 1.6 < 1.49（因为 49 > 6）
B. 1.6 > 1.49（整数相同，十分位 6 > 4）
C. 1.6 = 1.49（都是1点多）

🔬 深入探索

🤔 小数和分数的关系

每个有限小数都可以写成分数：
0.1 = 1/10（十分之一）
0.25 = 25/100 = 1/4（四分之一）
0.5 = 5/10 = 1/2（二分之一）
小数就是"以10的幂次为分母"的特殊分数！

🎉 课程总结

✅ 今天我们学到了

  • 🌡️ 小数是分数的"简写"：0.1 = 1/10
  • 📍 小数点是分界线：左边整数，右边小数
  • 📖 读法：整数部分正常读，小数点读"点"，小数部分逐位读
  • ⚖️ 比大小：先比整数，再比十分位，再比百分位……
