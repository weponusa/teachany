# 压强与浮力 · Pressure and Buoyancy

> 人教版八年级下册 · 第9-10章 · TeachAny 互动教学课件

## 📚 课件概述

本课件覆盖中学物理「压强与浮力」完整知识体系，包含 6 个教学模块：

| 模块 | 核心公式 | Canvas 实验 |
|:---|:---|:---|
| 压强基础 | p = F/S | 压力板方块实验 |
| 液体压强 | p = ρgh | 虚拟压强仪 |
| 大气压强 | p₀ ≈ 1.013×10⁵ Pa | 托里拆利模拟 |
| 浮力 | F浮 = F下 − F上 | 浮力压差可视化 |
| 阿基米德原理 | F浮 = ρ液gV排 | 弹簧秤溢水杯验证 |
| 浮沉条件 | 比较 ρ物 与 ρ液 | 浮沉条件模拟器 |

## ✨ 教学特色

- **ABT 叙事引导**：每个模块以 And-But-Therefore 结构引入

- **ConcepTest**：6 道即时检测题，每个选项含独立错因诊断
- **6 个 Canvas 互动实验**：HiDPI 支持，参数可调
- **脚手架分级练习**：Level 1（引导填空）→ Level 2（给提示）→ Level 3（独立设计）
- **10 题后测**：含 Bloom 层级覆盖分析和详细错因反馈
- **TTS 语音讲解**：7 段 MP3 音频 + 字幕系统 + 固定底部播放器
- **学习系统 v5.2**：进度追踪、SM-2 间隔复习、成就解锁
