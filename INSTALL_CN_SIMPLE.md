# 📦 TeachAny 一键安装指南（最简单）

> **适用对象**：国内中小学教师，无需翻墙，无需下载，**只需一个链接**

---

## 🚀 一键安装（推荐）

### 方法：复制链接 → 粘贴到 CodeBuddy

**安装链接**（Gitee 镜像，国内高速）：
```
https://gitee.com/wepon/teachany-opensource
```

### 安装步骤（30秒完成）

1. **复制上面的链接**（Ctrl+C 或 Command+C）

2. **打开 CodeBuddy/WorkBuddy 应用**

3. **输入安装命令**：
   ```
   /install-skill https://gitee.com/wepon/teachany-opensource
   ```
   或者在 **Skills 管理** 界面：
   - 点击 **"从 URL 安装"**
   - 粘贴链接
   - 点击 **"安装"**

4. **等待安装完成**（约10-30秒）

5. **开始使用**：
   ```
   /teachany 请生成"一次函数的应用"的课件
   ```

## 🆘 遇到问题？

### 问题1：安装失败，提示"网络错误"

**解决方案**：
1. 检查网络连接是否正常
2. 关闭 VPN/代理
3. 尝试切换到备用链接
4. 联系管理员检查防火墙设置

### 问题2：安装后找不到 Skill

**解决方案**：
1. 重启 CodeBuddy/WorkBuddy 应用
2. 在 Skills 管理中点击 **"刷新列表"**
3. 检查安装日志是否有错误提示

### 问题3：如何卸载？

在 Skills 管理界面：
1. 找到 **"TeachAny"**
2. 点击右侧的 **"卸载"** 按钮
3. 确认删除

## 🎓 教育理念

TeachAny 的核心理念：

1. **真实场景驱动**：每个知识点都对应现实生活中的应用
2. **五镜头教学法**：提问→定义→拆解→类比→应用
3. **ABT 叙事框架**：背景（AND）→冲突（BUT）→结论（THEREFORE）
4. **课标对齐**：所有内容对齐 2022 版课程标准

**我们相信**：教育不是填鸭式灌输，而是激发学生的好奇心，帮助他们建立知识与现实的连接。
