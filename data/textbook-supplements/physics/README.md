## 🎯 数据特色

### 1. **phenomena_library.json**（日常物理现象库）
收录10个学生日常生活中能观察到的物理现象，帮助建立"物理就在身边"的认知：
- 虹吸现象（液体压强）
- 冰面滑行（摩擦力）
- 声音传播延迟（波速）
- 微波炉转盘（驻波）
- 静电火花（电荷转移）
- 彩虹形成（光的折射与反射）
- 热气球升空（浮力与热膨胀）
- 回声（声波反射）
- 磁悬浮列车（磁场力）
- 日食月食（光的直线传播）

**教学价值**: 通过日常现象引入物理概念，降低学习门槛，激发学习兴趣。

---

### 2. **real_world_scenarios.json**（真实应用场景）
收录8个物理原理在现实中的应用案例，强调物理的实用性：
- 骑自行车的位移与距离（矢量与标量）
- 安全带救命原理（牛顿第一定律/惯性）
- 购物车推动（F=ma）
- 彗星着陆（弱引力环境）
- 雷电延迟（光速vs声速）
- 静电火花（电荷转移）
- 参考系选择（相对运动）
- 微波炉驻波（电磁波）

**教学价值**: 回答学生"学这个有什么用"的疑问，建立物理与生活的联系。

---

### 3. **analogies.json**（类比记忆库）
收录8个经典类比，用熟悉的事物解释抽象的物理概念：
- 购物车类比（F=ma）
- 冰球与保龄球类比（惯性与质量）
- 水流与管道类比（电流与电阻）
- 人浪类比（波动传播）
- 登山爬楼类比（重力势能）
- 台球反弹类比（光的反射与折射）
- 秋千类比（简谐运动）
- 自行车发电灯类比（电磁感应）

**教学价值**: 降低抽象概念的理解难度，提供记忆锚点，避免死记硬背。

---

### 4. **visual_strategies.json**（可视化策略库）
收录10个物理教学中常用的图示设计模式，规范可视化表达：
- 受力分析图（自由体图）
- 电路图绘制规范
- 运动轨迹图（v-t图与x-t图）
- 波动传播示意图
- 光的反射与折射光路图
- 能量转化流程图
- 磁场线分布图
- 抛体运动轨迹分解图
- 凸透镜成像光路图
- 简谐运动图像（x-t图与v-t图）

**教学价值**: 提供标准化的图示设计模板，提高课件的专业性和可读性。

## 💡 使用建议

### 在TeachAny课件生成中的应用

#### 1️⃣ Phase 0.5（教材数据查阅）
当生成力学、电学、光学、热学、波动等物理课件时，AI会自动查阅本目录的数据文件，优先使用教材验证过的真实案例和类比。

#### 2️⃣ Phase 1（ABT引入场景设计）
- 引入场景优先级：`real_world_scenarios` > 知识图谱 > Web搜索
- 记忆锚点优先级：`analogies` > 知识图谱 > AI生成
- 日常现象优先级：`phenomena_library` > Web搜索

#### 3️⃣ Phase 2（可视化设计）
参考`visual_strategies.json`中的标准图示设计模式，确保图表符合物理教学规范。

## 📝 使用示例

### 示例1：生成"牛顿第二定律"课件

**Phase 0.5查阅**:
```json
// 从real_world_scenarios.json获取
{
  "scenario_id": "phys_scenario_002",
  "title": "购物车推动实验",
  "description": "空购物车vs装满商品的购物车，推动时需要的力不同..."
}
```

**Phase 1查阅**:
```json
// 从analogies.json获取
{
  "analogy_id": "physics_analogy_001",
  "concept": "力与运动的关系",
  "concrete_analogy": {
    "scenario": "购物车类比",
    "description": "推购物车就像让物体受力运动..."
  }
}
```

**最终课件**: 真实购物场景 + 购物车类比 + F=ma公式推导
