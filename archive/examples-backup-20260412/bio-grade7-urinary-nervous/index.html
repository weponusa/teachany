尿的形成——肾单位的奥秘
✅ 已掌握
🔑 记忆锚点

尿的形成两步走：滤过 → 重吸收

🔬 肾单位结构 + 尿的形成流程
📋 两步流程详解
1
肾小球的滤过

血液流过肾小球，像筛子一样，将水、无机盐、尿素、葡萄糖滤入肾小囊，形成原尿。
⚠️ 蛋白质和血细胞不能滤过（分子太大）。

2
肾小管的重吸收

原尿流经肾小管时，全部葡萄糖、大部分水和无机盐被重新吸收回血液。
剩余的水、无机盐、尿素 → 尿液（尿素浓度大幅升高）。

血浆/原尿/尿液成分对比
✅ 已掌握
📊 三栏对比表
成分 血浆 原尿 尿液
水 ✓ 有 ✓ 有 少量
无机盐 ✓ 有 ✓ 有 少量
葡萄糖 ✓ 有 ✓ 有 ✗ 无（重吸收）
尿素 少量 少量 ↑↑ 浓度大增
大分子蛋白质 ✓ 有 ✗ 无（太大） ✗ 无
血细胞 ✓ 有 ✗ 无（太大） ✗ 无

规律：血浆→原尿（少了蛋白质+血细胞）→尿液（又少了葡萄糖，尿素浓度大增）

📌 核心知识速查
🫘 泌尿系统：肾脏（形成）→ 输尿管（输送）→ 膀胱（储存）→ 尿道（排出）
🔬 尿的形成：肾小球滤过（水/盐/糖/尿素→原尿，蛋白质留下）→ 肾小管重吸收（糖/大部分水盐→血液）→ 尿液
⚡ 反射弧口诀：感 → 传 → 中 → 传 → 效（感受器·传入·中枢·传出·效应器）
🔔 反射分类：非条件（先天/脊髓）= 缩手/眨眼；条件（后天/大脑）= 望梅/铃声
