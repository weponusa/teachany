背景

花粉落到柱头上，最终两粒精子要完成两次受精，这在植物界独一无二。双受精是什么？子房和胚珠分别发育成什么？

📚 核心知识讲解

一、传粉与花粉管

花粉从雄蕊（花粉囊）传到雌蕊（柱头）上的过程叫传粉。

花粉落在柱头上后：花粉萌发 → 长出花粉管 → 花粉管穿过花柱、子房壁 → 进入胚珠（经珠孔）→ 释放两个精子

二、双受精（被子植物特有）

花粉管中的两个精子进入胚珠后：

  • 精子① + 卵细胞 → 受精卵（→ 发育为胚）
  • 精子② + 极核（2个） → 受精极核（→ 发育为胚乳）

这种两个精子同时完成两次受精的现象叫双受精，是被子植物特有的。

1. 双受精是指：
A. 一粒花粉与两个卵细胞受精
B. 两粒花粉分别与两个卵细胞受精
C. 两个精子，一个与卵细胞融合，一个与极核融合
D. 一个精子与两个极核先后融合
✅ C正确！双受精是被子植物特有的：精子①+卵细胞→受精卵（发育为胚）；精子②+两个极核→受精极核（发育为胚乳）。两次受精同时完成。
2. 桃子的果肉（果皮）是由什么发育而来的？
A. 花瓣
B. 胚珠
C. 子房壁
D. 花萼
✅ C正确！子房壁发育为果皮（桃子果肉就是果皮）。子房整体发育为果实，胚珠（桃核里的种子）发育为种子。注意：桃核外面的硬壳也是果皮的一部分（内果皮）。
3. 一朵花中有5个胚珠，受精后最多能发育成多少粒种子？
A. 1粒
B. 3粒
C. 5粒
D. 10粒
✅ C正确！每个胚珠受精后发育为1粒种子。5个胚珠全部受精，最多发育为5粒种子。整个子房发育为含5粒种子的果实（如豌豆荚）。

📋 本课小结

传粉与花粉管

  • 花粉→柱头
  • 花粉管穿过花柱
  • 进入胚珠释放精子

双受精

  • 精子①+卵细胞→受精卵
  • 精子②+极核→受精极核
  • 被子植物特有

发育对应

  • 子房壁→果皮
  • 胚珠→种子
  • 受精卵→胚

整体关系

  • 子房→果实
  • 受精极核→胚乳
  • 花→凋落

🧠 核心对应关系

子房壁 → 果皮  |  胚珠 → 种子  |  子房 → 果实

受精卵 → 胚  |  受精极核 → 胚乳

⭐ 双受精 = 被子植物特有，两次融合同时进行
