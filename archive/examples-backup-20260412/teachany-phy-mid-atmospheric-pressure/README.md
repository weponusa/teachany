# 大气压强 — TeachAny 互动课件

> 从马德堡半球到托里拆利：探索看不见的巨大力量

## 📋 课件概述

本课件面向初中八年级物理"大气压强"单元，采用 TeachAny 暗色主题框架 v5.2，包含 3 个教学模块、3 个 Canvas 互动实验、14 道诊断式测评题目。

## 🎯 三大知识模块

| 模块 | 核心内容 | 互动实验 |
|:---|:---|:---|
| 大气压的存在 | 产生原因、马德堡半球实验(1654)、标准大气压、覆杯实验 | 马德堡半球模拟：拖动滑块调节马匹数(1-16)，观察拉力与大气压力的对比 |
| 大气压的测量 | 托里拆利实验(1643)、p₀=ρgh推导、管子倾斜分析、水银vs水对比 | 托里拆利实验模拟：海拔滑块(0-8848m) + 倾斜角滑块(0-60°)，竖直高度不变可视化 |
| 气压计与应用 | 三种气压计、气压与海拔关系表、高原反应、8种生活应用 | 海拔-气压-沸点探测器：快速选择城市按钮 + 山脉剖面 + 双仪表盘 |

## ✨ 功能特性

- 🌙 **暗色主题** — 护眼深色界面，适合课堂投影和自学
- 📖 **ABT叙事** — And/But/Therefore 结构引导每个模块
- 🧪 **Canvas实验** — 3个基于 Canvas 2D 的高清互动实验（支持 HiDPI）
- 📊 **知识图谱** — SVG 三列布局：前序知识 → 核心子知识点 → 后续知识
- ❓ **ConcepTest** — 3道概念检测题，每个选项均有错误诊断反馈
- 📝 **后测** — 8道选择题，含 wrongDiag 诊断式反馈
- 🪜 **脚手架** — 3级进阶练习（基础计算→变式思考→迁移挑战）
- 🏆 **学习系统 v5.2** — 进度追踪 + SM-2 间隔重复 + 成就解锁
- 🔊 **TTS语音** — 4段讲解音频 + 字幕同步 + 自动推进
- 📱 **响应式** — 移动端适配

## 📐 核心公式

| 公式 | 说明 |
|:---|:---|
| p₀ = ρ_Hg × g × h = 13600 × 9.8 × 0.76 ≈ 1.013×10⁵ Pa | 标准大气压 |
| p = 101325 × e^(-alt/8500) | 气压随海拔变化（气压高度公式） |
| 沸点 ≈ 100 - alt/300 (°C) | 沸点与海拔的近似关系 |
