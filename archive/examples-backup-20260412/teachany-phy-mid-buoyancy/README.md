# 浮力 — TeachAny 互动课件

> 从压力差到阿基米德原理，揭秘"浮与沉"的物理密码

## 📋 课件概述

本课件面向初中八年级物理"浮力"单元，采用 TeachAny 暗色主题框架 v5.2，包含 3 个教学模块、3 个 Canvas 互动实验、14 道诊断式测评题目。

## 🎯 三大知识模块

| 模块 | 核心内容 | 互动实验 |
|:---|:---|:---|
| 浮力的产生 | 上下压力差→浮力，F浮=F下−F上，完全浸没后浮力不变 | 浮力压差可视化：拖动浸入深度，观察上下压力箭头和浮力合力 |
| 阿基米德原理 | Eureka的故事，F浮=ρ液gV排，完全浸没vs部分浸没 | 溢水杯验证实验：双滑块（物体密度+浸入比例），弹簧秤+收集杯实时计算 |
| 浮沉条件 | 三种状态（下沉/悬浮/漂浮），密度比较法，冰山90%在水下 | 浮沉模拟器：密度滑块+快速选择材料，球体浮沉动画+力的箭头 |

## 📐 核心公式

| 公式 | 说明 |
|:---|:---|
| F浮 = F下 − F上 | 浮力本质：上下压力差 |
| F浮 = ρ液gV排 | 阿基米德原理 |
| V排/V物 = ρ物/ρ液 | 漂浮时浸入比例 |
