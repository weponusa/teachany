🎭 为什么铁块沉而铁船浮？

[And] 把铁块扔进水里，它会直接沉底。铁的密度约 7800 kg/m³，远大于水。
[But] 可是航空母舰几万吨重，全身都是钢铁，却稳稳浮在海面上——这不矛盾吗？
[Therefore] 关键不在"材料是什么"，而在排开液体的体积有多大。浮力 = ρ液gV排，船的空心结构排开了巨量的水，所以浮力 > 重力 → 漂浮！

📖 模块一：浮力的产生 — F浮 = F下 − F上

🤔 液体中的"隐形推力"

[And] 液体内部各方向都有压强，深度越大压强越大（p=ρgh）。
[But] 把一个石块缓缓放入水中，你能感觉水在"托"它——这个向上的力从何而来？
[Therefore] 物体上下表面处于不同深度，下表面深度更大 → 压强更大 → 受到的压力更大 → F浮 = F下 − F上。

💡 核心知识：浮力的本质是压力差

浸在液体中的物体，其上下表面分别承受液体的压力：

上表面（深度 h₁）：F上 = p₁ · S = ρ液gh₁S   （向下）

下表面（深度 h₂）：F下 = p₂ · S = ρ液gh₂S   （向上）

F浮 = F下 − F上 = ρ液g(h₂−h₁)S = ρ液gV排

其中 (h₂−h₁) 就是物体浸入液体的高度，乘以截面积 S 就是排开液体的体积 V排。

⚠️ 高频易错：完全浸没后浮力与深度无关

完全浸没后，物体上下表面深度差 (h₂−h₁) = 物体自身高度 = 不变 → V排不变 → F浮 = ρ液gV排 = 常数。

所以金属块从水下1米沉到5米，浮力不变！很多同学以为"越深浮力越大"，这是错的。

🌊 哪些情况有浮力？

场景有无浮力原因
木块漂浮在水面✅ 有下表面在水中，上表面在水面以上
石块沉在水底✅ 有上下表面深度不同，有压力差
桥墩插在河底⚠️ 特殊底面紧贴河底，下方无水，则该面无液体压力；浮力减小甚至为零
气球在空气中✅ 有气体也能产生浮力，原理相同

📖 模块二：阿基米德原理 — F浮 = ρ液gV排

🤔 浮力到底等于多少？

[And] 浮力来自上下表面的压力差，可以用 F下−F上 来计算。
[But] 现实中物体形状千奇百怪——圆球、不规则石头、船体，逐面算压力根本不现实。
[Therefore] 古希腊的阿基米德发现了一个惊人简洁的规律：F浮 = ρ液gV排——只要知道排开液体的体积就行！

🏛️ 阿基米德的"尤里卡"时刻

公元前 3 世纪，叙拉古国王请阿基米德鉴定王冠是否掺假。阿基米德苦思无果，直到某天泡澡时发现：身体入水越多，溢出的水越多，自己感觉越"轻"。

他激动地跳出浴缸大喊 "Eureka!"（我发现了！）——溢出水的重力 = 浮力！

💡 核心知识

阿基米德原理：浸在液体（或气体）中的物体受到的浮力，等于它排开液体（或气体）所受的重力。

F浮 = G排 = ρ液 · g · V排

📌 公式拆解

• ρ液 — 液体密度（kg/m³），不是物体密度！

• g — 重力加速度（≈10 N/kg）

• V排 — 排开液体的体积，不一定等于物体体积

⚠️ 关键区分

• 完全浸没：V排 = V物

• 漂浮/部分浸没：V排 < V物

• 对气体同样成立（热气球原理）

💎 阿基米德原理的妙处

不管物体什么形状（方的、圆的、不规则的），不管什么材料（铁的、木的、塑料的），浮力只跟两个量有关：液体密度 ρ液 和 排开体积 V排。

这就是为什么 F浮=ρ液gV排 如此强大——它把复杂的压力差计算简化成了一个优雅的公式。

🧮 典型例题

体积 200 cm³ 的铝块完全浸没在水中（g 取 10 N/kg），求浮力。

解：
V排 = V物 = 200 cm³ = 200×10⁻⁶ m³ = 2×10⁻⁴ m³
F浮 = ρ水gV排 = 1000 × 10 × 2×10⁻⁴ = 2 N

📖 模块三：物体的浮沉条件

🤔 什么决定浮还是沉？

[And] 我们知道 F浮 = ρ液gV排，G物 = ρ物gV物。
[But] 木块浮在水面、铁块沉入水底、鸡蛋在盐水中悬浮——同样是在水里，为什么结果截然不同？
[Therefore] 关键是比较 F浮 和 G物 的大小。对于实心均匀物体，本质就是比较 ρ物 和 ρ液！

💡 核心知识：三种浮沉状态

力的关系密度关系运动状态V排 关系生活实例
F浮 < G物 ρ物 > ρ液 下沉 V排 = V物 铁块在水中、石头沉底
F浮 = G物 ρ物 = ρ液 悬浮 V排 = V物 鸡蛋在适当盐水中
F浮 > G物 ρ物 < ρ液 上浮 → 漂浮 漂浮时 V排 < V物 木块浮在水面、冰山

⚠️ "悬浮"与"漂浮"的区别

悬浮：物体完全浸没在液体中且静止，ρ物 = ρ液，V排 = V物。可以在液体内部任意深度停留。

漂浮：物体浮在液面上且静止，ρ物 < ρ液，V排 < V物。一部分露出液面。

两者都满足 F浮 = G物，但 V排 不同！

🚢 漂浮问题的特殊性

漂浮时 F浮 = G物，这是解漂浮类题目的核心等式。

推导：漂浮时 ρ液gV排 = ρ物gV物
→ V排/V物 = ρ物/ρ液
→ 浸入体积占总体积的比例 = 密度之比

例：冰的密度 ≈ 0.9×10³ kg/m³，水密度 = 1.0×10³ kg/m³
→ V排/V物 = 0.9/1.0 = 90%
→ 冰山有 90% 体积在水下，只露出 10%——"冰山一角"由此而来！

🔧 改变浮沉的方法

让物体上浮

• 减小 ρ物：空心化（船舶设计）

• 增大 ρ液：加盐（死海浮人）

• 增大 V排：展平形状（铝箔实验）

让物体下沉

• 增大 ρ物：注水压舱（潜水艇）

• 减小 ρ液：从盐水换成淡水

• 减小 V排：排气缩小体积

🌍 生活中的浮沉应用

🚢 轮船：钢铁空心化 → 排开大量水 → F浮 > G → 漂浮。载货越多，吃水线越深。

🐟 潜水艇：压载水舱注水/排水 → 改变自身重力 → 下潜/上浮/悬浮。

🎈 热气球：加热空气 → 气球内空气密度 < 外部 → 浮力 > 重力 → 升空。

🏊 游泳：吸气时肺部膨胀 → V排增大 → 浮力增大 → 更容易浮。

🧊 冰山：ρ冰 < ρ水 → 90% 在水下 → 航海大忌。

🥚 盐水浮蛋：加盐增大 ρ液 → 鸡蛋从沉底到悬浮到漂浮。

Level 3 迁移挑战

设计题：一艘船质量 2×10⁵ kg，在海水（ρ=1.03×10³ kg/m³）中漂浮。

① 浮力是多少？② 排开海水体积是多少？③ 从海水驶入淡水会上浮还是下沉？

① 漂浮 → F浮 = G = mg = 2×10⁵ × 10 = 2×10⁶ N

② V排 = F浮/(ρ海水g) = 2×10⁶/(1.03×10³×10) ≈ 194.2 m³

③ 淡水 ρ 更小（1000 < 1030），为保持 F浮=G 不变，需要 V排 增大 → 船身下沉一些（吃水线上升）
