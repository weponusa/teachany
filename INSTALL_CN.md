## 🚀 快速安装（3分钟完成）

### 方案一：Gitee 镜像安装（推荐）⭐

> **Gitee** 是国内的代码托管平台，访问速度快，无需翻墙

#### 步骤 1：打开 Gitee 镜像仓库

在浏览器中访问：
```
https://gitee.com/wepon/teachany-opensource
```

#### 步骤 2：下载完整代码包

1. 点击页面右上角的绿色按钮 **"克隆/下载"**
2. 选择 **"下载 ZIP"**
3. 解压到任意文件夹（建议：`D:\TeachAny\`）

#### 步骤 3：安装到 CodeBuddy

1. 打开 **CodeBuddy** 应用
2. 打开 **设置** → **Skills 管理**
3. 点击 **"导入本地 Skill"**
4. 选择刚才解压的文件夹中的 `skill/` 目录
5. 点击 **"安装"**

✅ **完成！** 现在可以在 CodeBuddy 中使用 `/teachany` 命令生成课件了。

### 方案三：国内 CDN 直链下载

> 使用国内服务器加速，下载速度快

#### 下载命令（在命令行/终端中执行）

**Windows PowerShell**：
```powershell
# 下载到当前目录
Invoke-WebRequest -Uri "https://cdn.example.com/teachany-latest.zip" -OutFile "teachany.zip"

# 解压
Expand-Archive -Path "teachany.zip" -DestinationPath "D:\TeachAny"
```

**macOS/Linux**：
```bash
# 下载到当前目录
curl -O https://cdn.example.com/teachany-latest.zip

# 解压
unzip teachany.zip -d ~/TeachAny
```

## 🎯 使用方法

### 生成一个完整课件

在 CodeBuddy 中输入：
```
/teachany 请生成"一次函数的应用"的课件
```

AI 会自动：
1. 从知识图谱中查找相关节点
2. 从教材数据库中匹配教学方法和真实场景
3. 使用"五镜头法"生成结构化课件
4. 输出为 HTML 格式，可直接在浏览器中打开

### 指定学科和年级

```
/teachany 请生成"光合作用"的课件（初中生物，七年级）
```

### 查看可用的教学方法

```
/teachany 历史学科有哪些教学方法？
```

## 🔧 常见问题

### Q1：为什么选择 Gitee 而不是 GitHub？

**A**：GitHub 在国内访问不稳定，经常需要翻墙。Gitee 是国内平台，访问速度快，稳定性好，更适合中小学教师使用。

### Q2：百度网盘的文件会实时更新吗？

**A**：百度网盘的压缩包每月更新一次。如果需要最新版本，建议使用 Gitee 镜像（实时同步 GitHub）。

### Q3：如何更新到最新版本？

**方法1（Gitee）**：
1. 访问 https://gitee.com/wepon/teachany-opensource
2. 重新下载 ZIP 并覆盖安装

**方法2（自动更新）**：
- 在 CodeBuddy 的 Skills 管理中，点击 **"检查更新"**

### Q4：这个 Skill 收费吗？

**A**：完全免费！基于 MIT 协议开源，可以自由使用、修改、分享。

### Q5：可以在没有网络的情况下使用吗？

**A**：**可以！** 安装后，所有教材数据都存储在本地，无需联网即可生成课件（但 AI 生成功能需要联网调用大模型）。

### Q6：支持哪些年级和学科？

**A**：目前支持：
- **小学**：数学、语文（拼音识字）
- **初中**：数学、物理、化学、生物、历史、地理、语文
- **高中**：数学、物理、化学、生物、历史、地理、语文

## 🎓 教育理念

TeachAny 的核心理念：

1. **真实场景驱动**：每个知识点都对应现实生活中的应用
2. **五镜头教学法**：提问→定义→拆解→类比→应用
3. **ABT 叙事框架**：背景（AND）→冲突（BUT）→结论（THEREFORE）
4. **课标对齐**：所有内容对齐 2022 版课程标准

**我们相信**：教育不是填鸭式灌输，而是激发学生的好奇心，帮助他们建立知识与现实的连接。
