## ⚠️ 如果没有看到 "GitHub Actions" 选项

可能是 Actions 被禁用了,需要先启用:

1. 访问: https://github.com/weponusa/teachany/settings/actions
2. 找到 "Actions permissions"
3. 选择 "Allow all actions and reusable workflows"
4. 保存
5. 回到 Pages 设置页面重新操作

## 🆘 遇到问题?

### 问题1: 找不到 "GitHub Actions" 选项
**解决**: 先启用 Actions (见上面 "⚠️" 部分)

### 问题2: 切换后 Actions 没有自动运行
**解决**:
1. 访问: https://github.com/weponusa/teachany/actions
2. 点击 "Deploy to GitHub Pages" workflow
3. 点击右上角 "Run workflow" 按钮
4. 选择 "main" 分支
5. 点击绿色 "Run workflow" 按钮

### 问题3: Actions 运行失败 (红色 ❌)
**解决**:
1. 访问: https://github.com/weponusa/teachany/settings/actions
2. 拉到底部 "Workflow permissions"
3. 选择 "Read and write permissions"
4. 勾选 "Allow GitHub Actions to create and approve pull requests"
5. 保存
6. 回到 Actions 页面重新运行 workflow
